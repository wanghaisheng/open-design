### 4. PRD评估与优化 (prd-evaluation) 🆕 新增

**来源文档：** 好玩吗.md

**适用场景：**
- 评估游戏PRD的质量和可行性
- 从"好玩"、"商业化"、"长期运营"三个维度评估游戏
- 识别PRD中的冲突点和短板
- 提供PRD优化建议

**核心功能：**
- 好玩度评估（核心玩法循环、玩法独特性、心流体验）
- 商业化潜力评估（目标用户画像、付费点设计、价值感）
- 长期运营潜力评估（系统可扩展性、社交系统、Live Ops接口）
- 识别最大冲突点和核心优势
- 提供PRD优化行动计划

**与现有技能关系：**
- 补充 `design-review` - 专注PRD评估而非设计质量检查
- 补充 `concept-evaluation` - 专注PRD评估而非概念评估
- 独立技能，因为PRD评估有独特的评估框架（好玩+商业化+长期运营）

**优先级：** 中

**状态：** 待创建

- 节奏与留存设计
- 对标标杆案例分析

**与现有技能关系：**
- 补充 `design-review` - 专注体验优化而非质量检查
- 补充 `system-design` - 专注体验纯度而非系统功能
- 独立技能，因为核心体验提纯有独特的理论框架和方法论
