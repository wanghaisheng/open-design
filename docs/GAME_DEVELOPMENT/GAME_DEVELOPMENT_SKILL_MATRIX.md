# Game Development Skill Matrix

## 概述

本文档列出了 game-development-role 下的所有 skill，按照功能领域进行分类组织，便于查找和理解 skill 之间的关系。

**总技能数：** 101 个

**最后更新：** 2025-04-25

---

## 技能使用建议

### 设计阶段技能组合

**核心设计：**
- system-design + core-loop-design + economy-design + level-design

**商业化设计：**
- economy-design + vip-system-design + gacha-system-design + balance-check

**叙事设计：**
- narrative-design + story-design + narrative-writing + dialogue-writing + script-review

**视觉设计：**
- concept-art + concept-art-generation + visual-exploration + reference-collection + style-consistency-check

### 开发阶段技能组合

**核心开发：**
- engine-development + gameplay-implementation + shader-development + network-programming

**平台开发：**
- unity-development / unreal-development / godot-development + pc-development / mobile-development / console-development

**工具开发：**
- tool-development + pipeline-integration + setup-engine

### 测试阶段技能组合

**全面测试：**
- test-strategy + automation-testing + functional-testing + performance-testing + playtest-report

**质量保证：**
- gate-check + release-checklist + launch-checklist + visual-qa + script-review

### 管理阶段技能组合

**项目管理：**
- project-planning + sprint-plan + progress-tracking + milestone-review + retrospective

**风险管理：**
- risk-management + scope-check + resource-allocation + estimate + team-coordination

## 技能优先级建议

### 高优先级（核心技能）

1. **system-design** - 系统设计是游戏开发的基础
2. **gameplay-implementation** - 玩法实现是游戏开发的核心
3. **test-strategy** - 测试策略确保游戏质量
