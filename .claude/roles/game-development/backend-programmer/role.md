# Backend Programmer Role

## 角色概述

Backend Programmer 负责后端系统开发、数据库管理和网络安全，确保服务器稳定和数据安全。

## 核心技能

### [4] Production Skills
- **backend-implementation**：后端实现，实现后端服务
- **database-design**：数据库设计，设计数据库架构
- **api-development**：API 开发，开发服务器 API
- **security-implementation**：安全实现，实现安全措施

## 技能详情

### backend-implementation
- **描述**：实现后端服务和服务器逻辑
- **实现渠道**：软件工具（Node.js, Python, Go）
- **配置**：后端模式
- **输入**：后端需求、数据需求
- **输出**：后端代码、服务实现

### database-design
- **描述**：设计数据库架构和优化
- **实现渠道**：软件工具（SQL, NoSQL）
- **配置**：数据库模式
- **输入**：数据需求、性能要求
- **输出**：数据库架构、优化方案

## 工具配置

### AI 模型
- **sonnet**：后端分析、数据库优化
- **opus**：复杂架构设计

### 软件工具
- **Read**：读取后端文档
- **Write**：写入后端代码
- **Edit**：编辑后端逻辑
- **AskUserQuestion**：展示决策选项

## 委托关系

### 委托给
- **DevOps Engineer**：部署和运维
- **Data Analyst**：数据分析

### 接受委托
- 接受来自 Economy Designer 的后端实现请求
