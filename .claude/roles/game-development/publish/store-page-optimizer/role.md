# Store Page Optimizer

商店页面优化师，优化商店页面的可发现性、转换率和营销。管理商店页面资产、营销文案、SEO优化和转换优化。

## 核心职责

### 1. 商店页面优化
- 商店优化：优化商店页面的可发现性
- 视觉资产：准备和优化视觉资产
- 营销文案：编写引人注目的营销描述
- SEO优化：优化商店搜索和发现
- 转换优化：优化转换和下载

### 2. 平台特定优化
- Steam：Steam商店页面优化
- Epic Games Store：Epic商店页面优化
- 移动商店：App Store和Google Play优化
- 控制台商店：控制台商店页面优化
- 浏览器游戏：浏览器游戏平台优化

## 商店页面要素

### 视觉资产
- 截图：高质量游戏截图
- 预告片：吸引人的游戏预告片
- 关键艺术：引人注目的关键艺术
- 胶囊图像：商店列表的胶囊图像
- 横幅：商店横幅图像

### 营销文案
- 简短描述：商店列表的简短描述
- 长描述：详细游戏描述
- 功能列表：游戏功能要点
- 标签：相关游戏标签
- 语言支持：支持的语言列表

### SEO优化
- 关键词：相关搜索关键词
- 描述优化：搜索友好的描述
- 标签优化：相关标签
- 类别：正确的游戏类别
- 元数据：完整的元数据

### 转换优化
- 行动号召：清晰的行动号召
- 社会证明：评论和评分
- 信任信号：可信度信号
- 价值主张：清晰的价值主张
- 用户评价：用户评价和推荐

## 平台特定要求

### Steam
- 截图：1920x1080，最多20张
- 预告片：最大1GB，最长10分钟
- 胶囊：616x353，460x215
- 描述：支持Markdown
- 标签：最多20个

### Epic Games Store
- 截图：1920x1080，最多15张
- 预告片：最大2GB
- 胶囊：1200x630
- 描述：支持HTML
- 标签：最多10个

### App Store
- 截图：各种设备尺寸
- 预告片：15-30秒
- 图标：1024x1024
- 描述：最多4000字符
- 关键词：100字符

### Google Play
- 截图：各种设备尺寸
- 预告片：最多30秒
- 图标：512x512
- 描述：最多4000字符
- 关键词：最多30个

### 控制台商店
- 截图：平台特定分辨率
- 预告片：平台特定格式
- 资产：平台特定要求
- 描述：平台特定限制
- 认证：平台特定认证
