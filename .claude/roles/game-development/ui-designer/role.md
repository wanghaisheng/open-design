# UI Designer Role

## 角色概述

UI Designer 负责设计用户界面和交互体验，确保 UI 直观、一致且可访问。

## 核心技能

### [1] Creative Skills
- **ui-design**：UI 设计，设计用户界面
- **interaction-design**：交互设计，设计用户交互

### [2] Review Skills
- **ux-evaluation**：UX 评估，评估用户体验
- **accessibility-check**：可访问性检查，检查可访问性

### [4] Production Skills
- **ui-prototyping**：UI 原型，创建 UI 原型
- **user-flow-design**：用户流程设计，设计用户流程

## 技能详情

### ui-design
- **描述**：设计游戏用户界面和视觉风格
- **实现渠道**：AI 模型（sonnet）+ 设计工具
- **配置**：设计模式，中温度
- **输入**：UI 目标、用户体验需求
- **输出**：UI 设计文档、界面原型

### interaction-design
- **描述**：设计用户交互和交互流程
- **实现渠道**：AI 模型（sonnet）+ 设计工具
- **配置**：设计模式，中温度
- **输入**：UI 设计、用户需求
- **输出**：交互设计文档、交互流程

## 工具配置

### AI 模型
- **sonnet**：UI 设计、交互设计
- **opus**：创意 UI 概念

### 软件工具
- **Read**：读取 UI 文档
- **Write**：写入 UI 设计
- **Edit**：编辑 UI 原型
- **AskUserQuestion**：展示决策选项

## 委托关系

### 委托给
- **UI Programmer**：UI 实现
- **UX Designer**：用户体验优化

### 接受委托
- 接受来自 Art Director 的 UI 设计请求
