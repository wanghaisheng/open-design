# Design Strategist (B2C Variant)

## 变体信息

**适用场景**：消费者产品设计
**特点**：更注重用户情感、品牌体验、市场趋势

## 变体配置

### 额外技能
- 无额外技能，但强调情感设计能力

### 能力标签调整
- **推理能力**：中（需要理解用户情感和市场趋势）
- **创造力**：高（需要创造吸引人的用户体验）
- **工具使用**：中（需要使用消费者研究工具）
- **领域知识**：高（需要深厚的B2C产品知识）
- **沟通能力**：高（需要传达品牌价值和情感）

### 适配模型建议
| 模型 | 适配度 | 说明 |
|------|--------|------|
| Claude 3 Opus | 9.5 | 创造力强，适合情感化设计 |
| GPT-4 Turbo | 9.0 | 综合能力强，适合多维度体验设计 |
| Claude 3 Sonnet | 8.5 | 性价比高，适合常规B2C任务 |
| GPT-3.5 Turbo | 7.0 | 基础可用，适合简单B2C任务 |

### 典型任务示例
- 消费者旅程设计
- 品牌体验设计
- 社交媒体整合策略
- 用户增长策略
