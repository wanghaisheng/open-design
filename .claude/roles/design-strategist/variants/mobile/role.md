# Design Strategist (Mobile Variant)

## 变体信息

**适用场景**：移动应用设计
**特点**：更注重移动场景、触控交互、性能约束

## 变体配置

### 额外技能
- 无额外技能，但强调移动特性理解

### 能力标签调整
- **推理能力**：中（需要理解移动场景和约束）
- **创造力**：高（需要创造流畅的移动体验）
- **工具使用**：高（需要使用移动原型和测试工具）
- **领域知识**：高（需要深厚的移动产品知识）
- **沟通能力**：中（需要简洁明了的沟通）

### 适配模型建议
| 模型 | 适配度 | 说明 |
|------|--------|------|
| Claude 3 Opus | 9.0 | 综合能力强，适合复杂移动场景 |
| GPT-4 Turbo | 9.0 | 视觉能力强，适合移动UI设计 |
| Claude 3 Sonnet | 8.5 | 性价比高，适合常规移动任务 |
| GPT-3.5 Turbo | 7.0 | 基础可用，适合简单移动任务 |

### 典型任务示例
- 移动场景设计
- 触控交互设计
- 移动性能优化策略
- 平台特性适配
