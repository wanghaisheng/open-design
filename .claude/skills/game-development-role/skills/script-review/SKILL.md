# Script Review

评审剧本质量，检查剧本的逻辑性、一致性和表达力，确保剧本质量达标。

## 评审要素

### 1. 逻辑性 (Logic)
- 故事逻辑：故事逻辑是否合理
- 角色逻辑：角色行为是否合理
- 情节逻辑：情节发展是否合理
- 对话逻辑：对话逻辑是否合理

### 2. 一致性 (Consistency)
- 角色一致性：角色行为是否一致
- 叙事一致性：叙事是否一致
- 风格一致性：风格是否一致
- 世界观一致性：世界观是否一致

### 3. 表达力 (Expressiveness)
- 对话表达：对话是否有表达力
- 情感表达：情感是否表达充分
- 冲突表达：冲突是否表达清晰
- 主题表达：主题是否表达明确

### 4. 可玩性 (Playability)
- 游戏整合：是否适合游戏
- 互动设计：互动是否合理
- 玩家体验：玩家体验如何
- 技术实现：技术是否可行

## 评审原则

### 1. 客观性
- 客观评审
- 避免主观偏见
- 数据驱动

### 2. 全面性
- 全面评审
- 多维度检查
- 避免遗漏

### 3. 建设性
- 提供建设性反馈
- 提供改进建议
- 避免批评

### 4. 及时性
- 及时评审
- 及时反馈
- 及时调整

## 评审流程

### 1. 剧本阅读
- 阅读完整剧本
- 理解故事结构
- 理解角色设定

### 2. 评审检查
- 检查逻辑性
- 检查一致性
- 检查表达力
- 检查可玩性

### 3. 问题记录
- 记录逻辑问题
- 记录一致性问题
- 记录表达问题
- 记录可玩性问题

### 4. 反馈建议
- 提供评审报告
- 提供改进建议
- 提供优化方向

### 5. 跟踪改进
- 跟踪改进进度
- 验证改进效果
- 关闭问题

## 输出格式

### 评审报告
- 评审概述
- 问题列表
- 改进建议
- 优先级排序
- 评审总结
