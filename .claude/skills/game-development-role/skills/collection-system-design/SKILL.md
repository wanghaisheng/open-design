## 操作步骤

### 步骤1：读取PRD和收集需求

**目标**：读取游戏PRD，理解收集系统需求和目标

**操作方法**：
1. 读取游戏PRD文档
2. 提取游戏类型和核心玩法
3. 提取目标用户和收集偏好
4. 提取现有收集系统设计
5. 分析收集系统的核心驱动力
6. 提取商业化目标和付费收集需求

**检查清单**：

## 操作注意事项

### 核心设计原则

1. **收集癖驱动**：收集系统应利用玩家的收集癖心理，驱动持续收集
2. **完成度激励**：收集系统应提供清晰的完成度激励，让玩家有明确目标
3. **长期留存**：收集系统应支持长期留存，提供持续收集目标
4. **平衡性**：收集系统应平衡免费和付费玩家体验

### 稀有度设计原则

1. **稀有度分层**：稀有度应有明显分层，让稀有收集品更有价值
2. **概率透明**：稀有度获取概率应透明，避免玩家质疑
3. **保底机制**：应设计保底机制，避免玩家长期无法获得稀有收集品
4. **稀有度平衡**：稀有度应保持平衡，避免稀有收集品过于影响游戏平衡

### 获取途径设计原则

1. **多样性**：提供多样的获取途径，满足不同玩家偏好
2. **公平性**：免费和付费获取途径应保持公平
3. **挑战性**：稀有收集品获取应有挑战性
4. **可持续性**：获取途径应可持续，支持长期收集

### 进度系统设计原则

1. **清晰性**：进度显示应清晰，让玩家一目了然
2. **激励性**：进度激励应有吸引力，驱动玩家继续收集
3. **里程碑**：应设置里程碑，提供阶段性奖励
4. **社交性**：进度应可分享，支持社交炫耀

### 付费平衡原则

1. **公平性**：付费不应破坏收集系统的公平性
2. **加速而非必需**：付费应加速收集而非必需
3. **价值感**：付费收集品应有明确价值感
4. **免费路径**：提供完整的免费收集路径

## 常见错误

### 错误1：稀有度设计不合理

**问题**：稀有度分层不明显，或稀有收集品过于影响平衡

**修正**：设计明显的稀有度分层，保持稀有收集品的平衡性

### 错误2：获取途径过于单一

**问题**：获取途径单一，缺乏多样性

**修正**：提供多样的获取途径，满足不同玩家偏好

### 错误3：进度系统不清晰

**问题**：进度显示不清晰，玩家无法了解收集进度

**修正**：设计清晰的进度显示，让玩家一目了然

### 错误4：收集奖励不足

**问题**：收集奖励不足，缺乏吸引力

**修正**：设计有吸引力的收集奖励，驱动玩家持续收集

### 错误5：付费影响公平性

**问题**：付费严重破坏收集系统的公平性

**修正**：确保付费加速而非必需，保持收集系统的公平性

### 错误6：图鉴系统不完善

**问题**：图鉴系统不完善，玩家无法查看收集品信息

**修正**：设计完善的图鉴系统，提供详细的收集品信息
