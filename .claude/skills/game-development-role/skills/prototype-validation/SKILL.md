# Prototype Validation

原型验证技能，使用四维框架（趋势、受众、核心玩法、变现）验证原型潜力。

## 适用场景

- 验证游戏原型潜力
- 评估结构性潜力
- 加速止损决策
- 避免无效开发
- 混合休闲游戏验证

## 操作步骤

### 步骤1：趋势维度验证

**目标**：验证原型是否体现低CPI吸量潜力

**操作方法**：
1. **广告素材潜力评估**
   - 评估能否制作低CPI广告素材
   - 设定目标CPI（如<$0.35）
   - 分析广告创意空间
   - 评估素材制作难度

2. **趋势匹配度分析**
   - 分析原型与市场趋势匹配度
   - 评估题材新颖性
   - 分析竞争情况
   - 识别差异化机会

3. **玩法融合评估**
   - 评估玩法融合是否自然
   - 分析融合创新性
   - 评估融合可行性
   - 识别融合风险

**检查清单**：
- [ ] 广告潜力已评估
- [ ] 趋势匹配已分析
- [ ] 玩法融合已评估
- [ ] 趋势维度已验证

### 步骤2：受众维度验证

**目标**：验证目标用户定义是否清晰

**操作方法**：
1. **用户画像定义**
   - 定义目标用户性别
   - 定义目标用户年龄
   - 定义目标用户地区
   - 定义用户行为特征

2. **市场选择分析**
   - 分析目标市场特征
   - 评估市场规模
   - 分析市场收入占比
   - 选择核心市场

3. **驱动力分析**
   - 分析用户核心驱动力
   - 确定IAP驱动还是IAA驱动
   - 分析付费意愿
   - 分析游戏习惯

**检查清单**：
- [ ] 用户画像已定义
- [ ] 市场已选择
- [ ] 驱动力已分析
- [ ] 受众维度已验证

### 步骤3：核心玩法维度验证

**目标**：验证玩法是否支持规模化生产

**操作方法**：
1. **关卡量产能力评估**
   - 评估能否设计100+关卡
   - 分析关卡设计难度
   - 评估关卡多样性
   - 分析关卡成本

2. **难度曲线设计**
   - 设计可控难度曲线
   - 分析难度递进
   - 评估难度平衡
   - 设计难度调节机制

3. **融合机制分析**
   - 分析融合机制自然性
   - 评估融合深度
   - 分析融合风险
   - 优化融合方案

**检查清单**：
- [ ] 关卡量产已评估
- [ ] 难度曲线已设计
- [ ] 融合机制已分析
- [ ] 核心玩法维度已验证

### 步骤4：变现维度验证

**目标**：验证变现路径是否清晰

**操作方法**：
1. **变现模式选择**
   - 明确主导模式（Hybrid/IAP/IAA）
   - 分析模式适配性
   - 评估模式可行性
   - 选择最佳模式

2. **变现路径设计**
   - 设计核心付费点
   - 设计广告位
   - 设计资源循环
   - 设计变现节奏

3. **经济系统验证**
   - 验证经济系统平衡
   - 分析资源产出消耗
   - 评估经济可持续性
   - 优化经济设计

**检查清单**：
- [ ] 变现模式已选择
- [ ] 变现路径已设计
- [ ] 经济系统已验证
- [ ] 变现维度已验证

### 步骤5：综合评估与决策

**目标**：基于四维框架做出决策

**操作方法**：
1. **结构性潜力评估**
   - 评估四维度整体表现
   - 识别结构性短板
   - 评估改进可行性
   - 判断结构性潜力

2. **决策制定**
   - **通过验证**：继续开发
   - **部分通过**：允许一次修复性测试
   - **不通过**：立即中止（KILL SWITCH）

3. **决策记录**
   - 记录评估过程
   - 记录决策理由
   - 记录改进建议
   - 归档决策文档

**检查清单**：
- [ ] 结构性潜力已评估
- [ ] 决策已制定
- [ ] 决策已记录
- [ ] 验证已完成

## 操作注意事项

### 验证原则

1. **结构性视角**：从结构性潜力角度评估
2. **加速止损**：果断中止无潜力项目
3. **数据驱动**：用数据和事实说话
4. **早期决策**：在原型阶段就做出决策

### 四维平衡

1. **全面评估**：四维度都要评估
2. **短板识别**：识别结构性短板
3. **优先级排序**：按重要性排序维度
4. **综合判断**：综合四维度判断

### 决策执行

1. **果断决策**：基于评估果断决策
2. **记录完整**：完整记录决策过程
3. **及时沟通**：及时沟通决策结果
4. **执行坚决**：坚决执行决策

## 常见错误

### 错误1：只看数据

**问题**：只看测试数据，忽视结构性潜力

**修正**：从四维框架评估结构性潜力

### 错误2：不舍得中止

**问题**：对项目不舍得中止，继续投入

**修正**：果断中止无潜力项目，加速止损

### 错误3：维度不全

**问题**：只评估部分维度，评估不全面

**修正**：完整评估四维度，全面判断

### 错误4：决策随意

**问题**：决策随意，缺乏依据

**修正**：基于充分评估和数据做出决策

### 错误5：不记录决策

**问题**：不记录决策过程，难以追溯

**修正**：完整记录决策过程和理由
