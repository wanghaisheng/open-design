# Indoor Environment

室内环境设计技能，创建游戏中的室内场景、建筑结构、家具和室内照明。

## 适用场景

- 设计室内游戏场景
- 创建建筑结构和空间
- 添加家具和装饰元素
- 实现室内照明效果
- 优化室内环境性能

## 操作步骤

### 步骤1：空间设计

**目标**：设计室内空间布局和结构

**操作方法**：
1. 确定房间功能和用途
2. 设计房间尺寸和比例
3. 创建墙体、地板、天花板
4. 设计门窗和开口
5. 实现楼梯和多层结构

**检查清单**：
- [ ] 房间功能已确定
- [ ] 尺寸和比例已设计
- [ ] 基础结构已创建
- [ ] 门窗已添加
- [ ] 多层结构已实现

### 步骤2：家具布置

**目标**：添加家具和装饰元素

**操作方法**：
1. 选择合适的家具资源
2. 布置家具位置和朝向
3. 实现家具互动（可打开、可使用）
4. 添加装饰元素（画作、地毯、灯具）
5. 优化家具碰撞体

**检查清单**：
- [ ] 家具资源已选择
- [ ] 家具已布置
- [ ] 互动已实现
- [ ] 装饰已添加
- [ ] 碰撞体已优化

### 步骤3：材质和纹理

**目标**：应用材质和纹理

**操作方法**：
1. 选择墙面材质（砖、木、混凝土）
2. 选择地板材质（瓷砖、木地板、地毯）
3. 实现材质混合和细节
4. 添加法线贴图和粗糙度贴图
5. 实现材质磨损和污渍

**检查清单**：
- [ ] 墙面材质已选择
- [ ] 地板材质已选择
- [ ] 材质混合已实现
- [ ] 贴图已添加
- [ ] 磨损效果已实现

### 步骤4：室内照明

**目标**：设计室内照明效果

**操作方法**：
1. 设置主光源（吊灯、窗户光）
2. 添加辅助光源（台灯、壁灯）
3. 实现光照衰减和阴影
4. 添加体积光和光晕
5. 实现灯光开关和调光

**检查清单**：
- [ ] 主光源已设置
- [ ] 辅助光源已添加
- [ ] 光照衰减已实现
- [ ] 体积光已添加
- [ ] 灯光控制已实现

### 步骤5：环境细节

**目标**：添加环境细节和生活气息

**操作方法**：
1. 添加日常物品（书籍、杯子、餐具）
2. 实现物品摆放的自然感
3. 添加环境音效（电视、时钟）
4. 实现环境互动（可移动物品）
5. 添加叙事元素（照片、信件）

**检查清单**：

### 步骤6：性能优化

**目标**：优化室内环境性能

**操作方法**：

## 操作注意事项

### 空间设计

1. **比例合理**：房间尺寸要符合人体工程学
2. **动线流畅**：玩家移动路径要顺畅
3. **视野开阔**：避免过于拥挤的空间

### 视觉一致性

1. **风格统一**：保持美术风格一致
2. **材质协调**：材质搭配要和谐
3. **色彩平衡**：使用统一的调色板

### 玩家体验

1. **导航清晰**：玩家能识别房间功能
2. **互动直观**：可互动物品要有视觉提示
3. **沉浸感强**：细节丰富，生活气息浓

## 常见错误

### 错误1：空间比例失调

**问题**：房间尺寸不合理，过大或过小

**修正**：参考真实建筑比例，考虑游戏视角

### 错误2：家具布置生硬

**问题**：家具摆放过于整齐，缺乏生活气息

**修正**：调整家具位置，添加日常物品

### 错误3：照明不自然

**问题**：室内光照过于均匀或过暗

**修正**：使用多种光源，实现光照层次

### 错误4：材质重复

**问题**：同一材质在多个房间重复使用

**修正**：为不同房间使用不同材质组合

### 错误5：性能问题

**问题**：室内场景卡顿，帧率低

**修正**：使用门户系统、光照烘焙、遮挡剔除优化性能
