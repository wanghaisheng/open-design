# Asset Generation

生成游戏资产，包括图像、模型、音频等，使用AI工具和传统工具创建高质量游戏资产。

## 适用场景

- 需要生成图像资产
- 需要生成3D资产
- 需要生成音频资产
- 需要生成其他资产
- 需要批量生成资产

## 操作步骤

### 步骤1：需求分析

**目标**：分析资产需求

**操作方法**：
1. 分析资产需求（功能、风格、用途）
2. 确定资产类型（图像、3D、音频、其他）
3. 确定质量标准（分辨率、格式、规格）

**检查清单**：
- [ ] 资产需求已分析
- [ ] 资产类型已确定
- [ ] 质量标准已确定

### 步骤2：工具选择

**目标**：选择合适的生成工具

**操作方法**：
1. 选择生成工具（AI工具或传统工具）
2. 评估成本效益
3. 确定工作流

**检查清单**：
- [ ] 生成工具已选择
- [ ] 成本效益已评估
- [ ] 工作流已确定

### 步骤3：资产生成

**目标**：生成资产

**操作方法**：
1. **图像资产**：使用 Gemini/Grok/Midjourney 生成，或使用 Photoshop/GIMP 创建
2. **3D资产**：使用 Tripo3D/Meshy 生成，或使用 Blender/Maya 创建
3. **音频资产**：使用 Suno/ElevenLabs 生成，或使用 Audacity/Reaper 创建
4. **其他资产**：使用 GPT/Claude 生成文本，或使用其他工具

**检查清单**：
- [ ] 图像资产已生成（如适用）
- [ ] 3D资产已生成（如适用）
- [ ] 音频资产已生成（如适用）
- [ ] 其他资产已生成（如适用）

### 步骤4：质量检查

**目标**：检查资产质量

**操作方法**：
1. 检查资产质量（分辨率、细节、完整性）
2. 检查风格一致性（与游戏整体风格一致）
3. 检查技术规范（格式、规格、命名）

**检查清单**：
- [ ] 资产质量已检查
- [ ] 风格一致性已检查
- [ ] 技术规范已检查

### 步骤5：优化调整

**目标**：优化和调整资产

**操作方法**：
1. 优化资产质量（提升细节、完善设计）
2. 调整资产风格（确保风格一致）
3. 优化资产性能（压缩、优化）

**检查清单**：
- [ ] 资产质量已优化
- [ ] 资产风格已调整
- [ ] 资产性能已优化

### 步骤6：交付管理

**目标**：管理资产交付

**操作方法**：
1. 资产命名规范（遵循命名规范）
2. 资产版本管理（版本控制）
3. 资产文档管理（编写使用说明）

**检查清单**：
- [ ] 资产命名规范已遵循
- [ ] 资产版本已管理
- [ ] 资产文档已编写

- 资产使用说明
- 资产版本历史

## 常见错误

### 错误1：不分析需求

**问题**：不分析资产需求，直接生成

**修正**：必须先分析资产需求，确定资产类型和质量标准

### 错误2：不选择合适工具

**问题**：不评估成本效益，随意选择工具

**修正**：必须评估成本效益，选择合适的生成工具

### 错误3：不检查风格一致性

**问题**：不检查资产风格是否与游戏整体风格一致

**修正**：必须检查风格一致性，确保与游戏整体风格一致

### 错误4：不遵循命名规范

**问题**：不遵循资产命名规范

**修正**：必须遵循资产命名规范

### 错误5：不进行版本管理

**问题**：不进行资产版本管理

**修正**：必须进行资产版本管理
