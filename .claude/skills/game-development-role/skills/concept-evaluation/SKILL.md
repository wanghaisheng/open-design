# Concept Evaluation

评估概念质量，从创意性、可行性、一致性和沟通性等多个维度评估视觉概念的质量。

## 评估维度

### 1. 创意性 (Creativity)
- 独特性：概念是否独特
- 创新性：概念是否有创新
- 视觉冲击：概念是否有视觉冲击
- 记忆点：概念是否有记忆点

### 2. 可行性 (Feasibility)
- 技术可行性：技术实现是否可行
- 资源可行性：资源是否充足
- 时间可行性：时间是否充足
- 成本可行性：成本是否可控

### 3. 一致性 (Consistency)
- 风格一致性：是否符合游戏风格
- 色彩一致性：色彩是否一致
- 氛围一致性：氛围是否一致
- 整体一致性：整体是否一致

### 4. 沟通性 (Communication)
- 清晰度：概念是否清晰
- 易理解性：是否易于理解
- 表达力：概念是否有表达力
- 传达效果：传达效果如何
