## 操作注意事项

### 协调机制

1. **流程清晰**：协调流程要清晰明确
2. **沟通及时**：及时沟通协调
3. **权限明确**：决策权限要明确
4. **记录完整**：完整记录协调过程

### 冲突解决

1. **及时处理**：及时处理冲突
2. **客观公正**：客观公正地解决冲突
3. **寻求共识**：寻求团队共识
4. **记录决策**：记录决策和理由

### 团队协作

1. **促进沟通**：促进团队间沟通
2. **建立信任**：建立团队信任
3. **共享信息**：共享美学信息
4. **共同目标**：确立共同目标

## 常见错误

### 错误1：协调不及时

**问题**：协调不及时，导致冲突扩大

**修正**：建立定期协调机制，及时处理冲突

### 错误2：决策权限不清

**问题**：决策权限不清，导致决策混乱

**修正**：明确决策权限和流程

### 错误3：不记录协调

**问题**：不记录协调过程，难以追溯

**修正**：完整记录协调过程和决策

### 错误4：忽视团队意见

**问题**：忽视团队意见，导致不满

**修正**：充分听取团队意见，寻求共识

### 错误5：不跟踪进度

**问题**：协调后不跟踪进度

**修正**：跟踪协调进度，确保执行
