# Character Consistency Check

检查角色一致性，确保角色在不同场景、不同对话中的行为和表达保持一致。

## 适用场景

- 需要检查角色性格一致性
- 需要检查角色背景一致性
- 需要检查对话一致性
- 需要检查视觉一致性
- 需要确保角色在不同场景中表现一致

## 操作步骤

### 步骤1：建立角色档案

**目标**：建立角色基准档案

**操作方法**：
1. 建立角色性格档案（性格特征、情绪反应、行为模式、决策方式）
2. 建立角色背景档案（背景故事、角色关系、角色动机、角色成长）
3. 建立角色对话档案（语言风格、说话习惯、对话内容、对话语气）
4. 建立角色视觉档案（外观设计、服装设计、动作设计、表情设计）

**检查清单**：
- [ ] 角色性格档案已建立
- [ ] 角色背景档案已建立
- [ ] 角色对话档案已建立
- [ ] 角色视觉档案已建立

### 步骤2：检查性格一致性

**目标**：检查角色性格表达是否一致

**操作方法**：
1. 检查性格表达是否一致
2. 检查情绪反应是否一致
3. 检查行为模式是否一致
4. 检查决策方式是否一致

- [ ] 决策方式已检查

### 步骤3：检查背景一致性

**目标**：检查角色背景是否一致

**操作方法**：
1. 检查背景故事是否一致
2. 检查角色关系是否一致
3. 检查角色动机是否一致
4. 检查角色成长是否合理

**检查清单**：

**目标**：检查对话是否一致

2. 检查说话习惯是否一致

**目标**：检查视觉表现是否一致

**操作方法**：
1. 检查外观设计是否一致
2. 检查服装设计是否一致
3. 检查动作设计是否一致
4. 检查表情设计是否一致

- [ ] 外观设计已检查

**目标**：记录发现的不一致问题

## 常见错误

### 错误1：不建立角色档案

**问题**：不建立角色基准档案，无法判断一致性

**修正**：必须先建立角色档案（性格、背景、对话、视觉）

### 错误2：只检查单一维度

**问题**：只检查对话一致性，不检查性格、背景、视觉一致性

**修正**：必须全面检查所有维度（性格、背景、对话、视觉）

### 错误3：不记录问题严重性

**问题**：记录问题但不评估严重性

**修正**：必须记录问题严重性（Critical/High/Medium/Low）

### 错误4：不跟踪修正进度

**问题**：提供检查报告后不跟踪修正进度

**修正**：必须跟踪修正进度，验证修正结果

### 错误5：不更新角色档案

**问题**：修正后不更新角色档案

**修正**：修正后必须更新角色档案
