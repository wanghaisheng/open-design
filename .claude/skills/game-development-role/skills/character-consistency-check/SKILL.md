# Character Consistency Check

检查角色一致性，确保角色在不同场景、不同对话中的行为和表达保持一致。

## 检查要素

### 1. 角色性格一致性 (Personality Consistency)
- 性格表达：角色性格表达是否一致
- 情绪反应：角色情绪反应是否一致
- 行为模式：角色行为模式是否一致
- 决策方式：角色决策方式是否一致

### 2. 角色背景一致性 (Background Consistency)
- 背景故事：角色背景故事是否一致
- 角色关系：角色关系是否一致
- 角色动机：角色动机是否一致
- 角色成长：角色成长是否合理

### 3. 对话一致性 (Dialogue Consistency)
- 语言风格：角色语言风格是否一致
- 说话习惯：角色说话习惯是否一致
- 对话内容：对话内容是否符合角色
- 对话语气：对话语气是否一致

### 4. 视觉一致性 (Visual Consistency)
- 外观设计：角色外观是否一致
- 服装设计：角色服装是否一致
- 动作设计：角色动作是否一致
- 表情设计：角色表情是否一致

## 检查原则

### 1. 严格性
- 严格检查标准
- 严格执行规则
- 严格质量要求

### 2. 全面性
- 全面检查角色
- 全面检查场景
- 全面检查对话

### 3. 及时性
- 及时检查
- 及时反馈
- 及时修正

### 4. 可追溯性
- 记录检查结果
- 记录检查过程
- 记录修正历史

## 检查流程

### 1. 建立角色档案
- 建立角色性格档案
- 建立角色背景档案
- 建立角色对话档案
- 建立角色视觉档案

### 2. 执行检查
- 检查性格一致性
- 检查背景一致性
- 检查对话一致性
- 检查视觉一致性

### 3. 问题记录
- 记录不一致问题
- 记录问题严重性
- 记录修正建议

### 4. 反馈修正

- 提供修正建议
- 跟踪修正进度
