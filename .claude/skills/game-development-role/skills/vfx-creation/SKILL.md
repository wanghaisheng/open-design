# VFX Creation

制作游戏视觉特效，包括粒子系统、着色器特效、后处理特效和物理模拟，创造震撼的视觉效果。

## 特效类型

### 1. 粒子系统 (Particle Systems)
- 粒子发射：粒子发射器
- 粒子运动：粒子运动模拟
- 粒子生命周期：粒子生命周期
- 粒子渲染：粒子渲染技术

### 2. 着色器特效 (Shader Effects)
- 顶点着色器：顶点变形
- 片段着色器：像素着色
- 计算着色器：GPU计算
- 后处理着色器：后期处理

### 3. 后处理特效 (Post-Processing Effects)
- 泛光：Bloom效果
- 景深：Depth of Field
- 运动模糊：Motion Blur
- 色调映射：Tone Mapping

### 4. 物理模拟 (Physics Simulation)
- 流体模拟：流体动力学
- 烟雾模拟：烟雾效果
- 火焰模拟：火焰效果
- 破坏模拟：破坏效果

- 自定义着色器：自定义后处理
