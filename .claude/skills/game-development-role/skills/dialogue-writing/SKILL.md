# Dialogue Writing

编写角色对话和台词，创造生动自然的角色对话，增强角色个性和故事表达。

## 对话要素

### 1. 角色声音 (Character Voice)
- 角色性格：角色性格表达
- 语言风格：语言风格设定
- 说话习惯：说话习惯设计
- 情绪表达：情绪表达方式

### 2. 对话内容 (Dialogue Content)
- 主要对话：主要情节对话
- 次要对话：次要情节对话
- 背景对话：背景设定对话
- 互动对话：玩家互动对话

### 3. 对话结构 (Dialogue Structure)
- 对话流程：对话流程设计
- 对话分支：对话分支设计
- 对话触发：对话触发条件
- 对话结束：对话结束条件

### 4. 对话质量 (Dialogue Quality)
- 自然性：对话自然流畅
- 一致性：对话与角色一致
- 表达力：对话表达力强
- 沉浸感：对话增强沉浸感

## 编写原则

### 1. 角色一致性
- 对话与角色一致
- 语言风格一致
- 性格表达一致

### 2. 自然流畅
- 对话自然流畅
- 避免生硬
- 符合日常习惯

### 3. 表达力强
- 对话表达力强
- 情感表达准确
- 信息传达清晰

### 4. 沉浸感
- 对话增强沉浸感
- 不打断游戏体验
- 符合游戏氛围

## 编写流程

### 1. 角色分析
- 分析角色性格
- 分析角色背景
- 分析角色关系

### 2. 对话设计
- 设计对话内容
- 设计对话结构
- 设计对话分支

### 3. 对话编写
- 编写对话内容
- 编写对话分支
- 编写对话触发

### 4. 对话优化
- 优化对话自然性
- 优化对话一致性
- 优化对话表达力

### 5. 对话测试
- 测试对话流程
- 测试对话分支
- 测试对话效果

## 输出格式

### 对话文档
- 对话内容
- 对话结构
- 对话分支
- 触发条件
- 对话测试报告
