# World Consistency Check

检查世界一致性，确保游戏世界的地理、历史、文化和规则在不同场景和内容中保持一致。

## 检查要素

### 1. 地理一致性 (Geography Consistency)
- 地理位置：地理位置是否一致
- 地形地貌：地形地貌是否一致
- 气候环境：气候环境是否一致
- 空间关系：空间关系是否一致

### 2. 历史一致性 (History Consistency)
- 历史事件：历史事件是否一致
- 时间线：时间线是否一致
- 历史影响：历史影响是否一致
- 历史人物：历史人物是否一致

### 3. 文化一致性 (Culture Consistency)
- 社会结构：社会结构是否一致
- 文化传统：文化传统是否一致
- 宗教信仰：宗教信仰是否一致
- 价值观念：价值观念是否一致

### 4. 规则一致性 (Rule Consistency)
- 物理规则：物理规则是否一致
- 魔法规则：魔法规则是否一致
- 社会规则：社会规则是否一致
- 经济规则：经济规则是否一致

## 检查原则

### 1. 严格性
- 严格检查标准
- 严格执行规则
- 严格质量要求

### 2. 全面性
- 全面检查世界
- 全面检查场景
- 全面检查内容

### 3. 及时性
- 及时检查
- 及时反馈
- 及时修正

### 4. 可追溯性
- 记录检查结果
- 记录检查过程
- 记录修正历史

## 检查流程

### 1. 建立世界档案
- 建立地理档案
- 建立历史档案
- 建立文化档案
- 建立规则档案

### 2. 执行检查
- 检查地理一致性
- 检查历史一致性
- 检查文化一致性
- 检查规则一致性

### 3. 问题记录
- 记录不一致问题
- 记录问题严重性
- 记录修正建议

### 4. 反馈修正
- 提供检查报告
- 提供修正建议
- 跟踪修正进度
