# Creative Discovery

创意发现技能，从三大支柱（成功模式转译、现实世界观察、玩法融合）中发现游戏创意。

## 适用场景

- 寻找混合休闲游戏创意
- 从成功模式中提取灵感
- 观察现实世界寻找题材
- 融合现有玩法创造新体验
- 降低获客成本（CPI）

## 操作步骤

### 步骤1：成功模式转译

**目标**：从已验证的成功模式中提取创意

**操作方法**：
1. **高概念转译（PC/主机 → 移动端）**
   - 识别PC/主机游戏的核心玩法
   - 提炼最吸引人的机制
   - 简化并适配移动端操作
   - 反复调试手感

2. **市场本土化改造（亚洲热门 → 西方市场）**
   - 分析亚洲市场成功游戏
   - 改造美术风格为欧美审美
   - 调整叙事节奏和UI/UX
   - 优化付费点设计

3. **内部玩法迭代（超休闲 → 超休闲）**
   - 对现有超休闲玩法换皮
   - 结合多种玩法微创新
   - 快速测试市场反应

**检查清单**：
- [ ] 核心玩法已提炼
- [ ] 移动端适配已完成
- [ ] 本土化改造已完成
- [ ] 微创新已实现

### 步骤2：现实世界观察

**目标**：从玩家已有认知中提取玩法

**操作方法**：
1. **挖掘通用体验（体育/童年游戏）**
   - 识别广为人知的运动
   - 提取核心规则
   - 抽象为触屏机制
   - 利用集体记忆

2. **捕捉网络潮流（病毒视频/满足感体验）**
   - 监控YouTube、TikTok
   - 寻找高播放量解压视频
   - 识别ASMR体验内容
   - 迅速游戏化

3. **发掘专业创意社区（Game Jam）**
   - 浏览Game Jam网站
   - 寻找未商业化原型
   - 识别闪光点创意
   - 二次开发打磨

**检查清单**：
- [ ] 通用体验已挖掘
- [ ] 网络潮流已捕捉
- [ ] 创意社区已浏览
- [ ] 创意已游戏化

### 步骤3：玩法融合与升维

**目标**：融合现有玩法创造混合休闲游戏

**操作方法**：
1. **核心公式应用**
   - 被验证的超休闲玩法 + 可扩展的RPG/策略元素 = 混合休闲
   - 找到低CPI超休闲核心玩法
   - 嫁接长线追求元系统

2. **分工明确设计**
   - 超休闲玩法：降低CPI、提供即时乐趣
   - RPG/策略元素：提升LTV、提供长线追求

3. **案例参考**
   - 《弓箭传说》：超休闲核心 + 数值成长系统
   - 《Cup Heroes》：弹球收集 + RPG英雄成长

**检查清单**：
- [ ] 核心公式已应用
- [ ] 分工设计已完成
- [ ] 案例已参考
- [ ] 融合方案已确定

## 操作注意事项

### 创意验证

1. **低门槛验证**：确保一眼看懂核心玩法
2. **拓展性验证**：确保能支撑长期系统
3. **市场验证**：参考成功案例和数据

### 融合原则

1. **自然融合**：避免生硬拼凑
2. **分工明确**：超休闲和深度系统各司其职
3. **渐进引入**：深度系统逐步引入

### 风险控制

1. **避免抄袭**：在借鉴基础上创新
2. **控制成本**：快速原型验证
3. **数据驱动**：用数据验证创意

## 常见错误

### 错误1：1:1复刻

**问题**：直接复刻成功游戏，缺乏创新

**修正**：提炼核心机制，进行差异化创新

### 错误2：忽视拓展性

**问题**：只关注低门槛，忽视长期拓展性

**修正**：在创意阶段就考虑系统深度潜力

### 错误3：生硬融合

**问题**：玩法融合不自然，体验割裂

**修正**：确保融合自然，各元素相互促进

### 错误4：过度复杂

**问题**：创意过于复杂，失去低门槛优势

**修正**：保持核心玩法简单，深度系统渐进

### 错误5：不验证

**问题**：创意不经过原型验证就投入开发

**修正**：快速原型验证，用数据决策
