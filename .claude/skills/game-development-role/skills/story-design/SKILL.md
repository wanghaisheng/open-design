# Story Design

故事设计，设计游戏故事结构、情节发展和叙事弧线。

## 适用场景

- 游戏故事结构设计
- 角色成长轨迹规划
- 世界观和背景设定
- 叙事弧线构建

## 操作步骤

### 步骤1：设计故事结构

**目标**：创建完整的叙事结构

**操作方法**：
1. 确定故事的核心冲突
2. 设计开端：建立背景和引入冲突
3. 规划发展：推进情节和角色成长
4. 构建高潮：冲突的顶点
5. 设计结局：解决冲突并给出结论

**检查清单**：
- [ ] 核心冲突明确
- [ ] 叙事结构完整
- [ ] 节奏合理
- [ ] 结局令人满意

### 步骤2：设计角色弧线

**目标**：创建角色成长轨迹

**操作方法**：
1. 定义角色的初始状态
2. 设计激励事件
3. 规划挑战与成长阶段
4. 设计转变点
5. 定义角色的最终状态

**检查清单**：
- [ ] 角色动机清晰
- [ ] 成长轨迹合理
- [ ] 转变点有意义
- [ ] 角色发展完整

### 步骤3：设计主题表达

**目标**：确保主题贯穿故事

**操作方法**：
1. 确定核心主题
2. 通过情节表达主题
3. 通过角色表达主题
4. 通过世界观表达主题
5. 确保主题一致性

**检查清单**：
- [ ] 主题明确
- [ ] 主题贯穿始终
- [ ] 表达方式多样
- [ ] 主题与游戏玩法相关

### 步骤4：设计世界背景

**目标**：创建故事发生的背景

**操作方法**：
1. 设计世界背景
2. 创建世界观和规则
3. 设计地理设定
4. 设计文化设定
5. 确保故事与世界观一致

**检查清单**：
- [ ] 世界观完整
- [ ] 规则一致
- [ ] 设定详细
- [ ] 与故事匹配

## 操作注意事项

### 一致性
- 保持故事内部逻辑一致
- 确保角色行为符合设定
- 维持世界观的一致性

### 参与度
- 创造引人入胜的情节
- 设计有吸引力的角色
- 建立情感连接

### 相关性
- 故事与游戏玩法相关
- 角色动机与玩家目标一致
- 主题与游戏体验匹配

## 输出格式

### 故事大纲
- 故事概要
- 主要角色
- 情节大纲
- 主题说明

### 角色档案
- 角色背景
- 角色目标
- 角色弧线
- 角色关系

### 世界设定
- 世界背景
- 世界规则
- 地理设定
- 文化设定

## 协作协议

### 向谁汇报
- narrative-director：叙事方向和审批
- creative-director：创意方向

### 协调对象
- writer：剧本编写
- level-designer：关卡中的叙事实现
- world-builder：世界中的叙事元素

## 常见错误

### 错误1：故事与玩法脱节
**问题**：故事与游戏玩法没有关联
**修正**：确保故事与玩法相互支持

### 错误2：角色动机不清晰
**问题**：角色的行为缺乏合理的动机
**修正**：明确角色的内在和外在动机

### 错误3：主题表达不明确
**问题**：主题过于模糊或分散
**修正**：聚焦核心主题，确保表达清晰
