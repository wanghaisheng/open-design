# Shader Development

开发着色器和材质，包括顶点着色器、片段着色器、计算着色器和后处理着色器，创造高质量的视觉效果。

## 着色器类型

### 1. 顶点着色器 (Vertex Shaders)
- 顶点变换：顶点坐标变换
- 法线变换：法线变换
- 纹理坐标：UV坐标处理
- 顶点动画：顶点动画效果

### 2. 片段着色器 (Fragment Shaders)
- 光照计算：光照模型
- 纹理采样：纹理采样
- 颜色混合：颜色混合
- 透明度处理：透明度计算

### 3. 计算着色器 (Compute Shaders)
- GPU计算：并行计算
- 粒子模拟：粒子计算
- 物理模拟：物理计算
- 后处理计算：后处理计算

### 4. 后处理着色器 (Post-Processing Shaders)
- 泛光：Bloom效果
- 景深：Depth of Field
- 色调映射：Tone Mapping
- 屏幕空间效果：SSAO, SSR

## 着色器语言

### 主要语言
- HLSL：DirectX着色器语言
- GLSL：OpenGL着色器语言
- Metal：Metal着色器语言
- WGSL：WebGPU着色器语言
