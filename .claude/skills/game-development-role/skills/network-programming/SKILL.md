# Network Programming

## 网络架构

### 1. 客户端-服务器 (Client-Server)
- 服务器架构：专用服务器、权威服务器
- 客户端逻辑：预测、插值、 reconciliation
- 网络协议：TCP, UDP, WebSocket
- 数据压缩：数据压缩算法

### 2. P2P (Peer-to-Peer)
- P2P架构：主机选择、连接建立
- 同步机制：状态同步、帧同步
- 网络拓扑：星型、网状
- 延迟处理：延迟补偿

### 3. 混合架构 (Hybrid)
- 混合架构：结合CS和P2P
- 负载均衡：服务器负载均衡
- 故障恢复：故障转移
- 扩展性：水平扩展
