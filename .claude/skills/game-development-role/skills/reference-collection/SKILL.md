# Reference Collection

收集视觉参考，收集游戏美术相关的视觉参考、艺术风格和灵感素材，为创作提供参考依据。

## 适用场景

- 需要收集视觉参考
- 需要收集艺术风格
- 需要收集灵感素材
- 需要创建情绪板
- 需要建立参考库

## 操作步骤

### 步骤1：需求分析

**目标**：分析参考需求

**操作方法**：
1. 分析参考需求（参考类型、参考数量、参考用途）
2. 确定参考方向（视觉风格、艺术风格、目标受众）
3. 确定参考类型（游戏参考、艺术参考、现实参考）

**检查清单**：
- [ ] 参考需求已分析
- [ ] 参考方向已确定
- [ ] 参考类型已确定

### 步骤2：搜索收集

**目标**：搜索和收集参考

**操作方法**：
1. 搜索相关参考（在线平台、本地资源）
2. 收集视觉素材（游戏参考、艺术参考、现实参考）
3. 收集风格参考（艺术风格、时代风格、地域风格）

**检查清单**：
- [ ] 相关参考已搜索
- [ ] 视觉素材已收集
- [ ] 风格参考已收集

### 步骤3：整理分类

**目标**：整理和分类参考

**操作方法**：
1. 分类整理参考（按类别、按风格、按用途）
2. 添加标签（标签管理、标签分类）
3. 创建情绪板（情绪板创建、情绪板组织）

**检查清单**：
- [ ] 参考已分类整理
- [ ] 标签已添加
- [ ] 情绪板已创建

### 步骤4：参考分析

**目标**：分析参考素材

**操作方法**：
1. 分析参考风格（风格特点、风格元素）
2. 分析参考元素（色彩、造型、构图）
3. 提取灵感（灵感提取、灵感应用）

**检查清单**：
- [ ] 参考风格已分析
- [ ] 参考元素已分析
- [ ] 灵感已提取

### 步骤5：参考分享

**目标**：分享参考资源

**操作方法**：
1. 分享参考库（参考库分享、参考库访问）
2. 分享情绪板（情绪板分享、情绪板说明）
3. 提供参考建议（使用建议、应用建议）

**检查清单**：
- [ ] 参考库已分享
- [ ] 情绪板已分享
- [ ] 参考建议已提供

## 常见错误

### 错误1：不分析参考需求

**问题**：不分析参考需求、不确定参考方向

**修正**：必须分析参考需求，确定参考方向，确定参考类型

### 错误2：不搜索相关参考

**问题**：不搜索相关参考、不收集视觉素材

**修正**：必须搜索相关参考，收集视觉素材，收集风格参考

### 错误3：不整理分类

**问题**：不分类整理参考、不添加标签

**修正**：必须分类整理参考，添加标签，创建情绪板

### 错误4：不分析参考

**问题**：不分析参考风格、不分析参考元素

**修正**：必须分析参考风格，参考元素，提取灵感

### 错误5：不分享参考

**问题**：不分享参考库、不提供参考建议

**修正**：必须分享参考库，分享情绪板，提供参考建议
