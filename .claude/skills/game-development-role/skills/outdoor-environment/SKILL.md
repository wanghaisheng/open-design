# Outdoor Environment

户外环境设计技能，创建游戏中的户外场景、地形、植被和天气效果。

## 适用场景

- 设计户外游戏场景
- 创建地形和地貌
- 添加植被和自然元素
- 实现天气和光照效果
- 优化户外环境性能

## 操作步骤

### 步骤1：地形设计

**目标**：创建游戏地形和地貌

**操作方法**：
1. 使用地形编辑工具创建基础地形
2. 设计山脉、丘陵、平原等地貌
3. 实现地形高度图和纹理混合
4. 添加河流、湖泊等水体
5. 优化地形网格和LOD

**检查清单**：
- [ ] 基础地形已创建
- [ ] 地貌已设计
- [ ] 地形纹理已混合
- [ ] 水体已添加
- [ ] 地形LOD已优化

### 步骤2：植被设计

**目标**：添加植被和自然元素

**操作方法**：
1. 选择合适的植被资源（树木、草、花）
2. 实现植被分布算法
3. 添加植被动画（风吹效果）
4. 优化植被渲染（实例化、遮挡剔除）
5. 实现植被季节变化

**检查清单**：
- [ ] 植被资源已选择
- [ ] 分布算法已实现
- [ ] 植被动画已添加
- [ ] 渲染已优化
- [ ] 季节变化已实现

### 步骤3：天气系统

**目标**：实现动态天气效果

**操作方法**：
1. 设计天气类型（晴天、雨天、雪天、雾天）
2. 实现天气过渡效果
3. 添加粒子效果（雨、雪、雾）
4. 实现光照和阴影变化
5. 同步天气与音效

**检查清单**：
- [ ] 天气类型已设计
- [ ] 过渡效果已实现
- [ ] 粒子效果已添加

- [ ] 音效已同步

### 步骤4：光照设计

**目标**：设计户外场景的光照

**操作方法**：
1. 设置主光源（太阳/月亮）
2. 实现昼夜循环
3. 添加环境光和反射光
4. 实现阴影（实时或烘焙）
5. 优化光照性能

### 步骤5：环境细节

**目标**：添加环境细节和装饰

**操作方法**：
1. 添加岩石、石头等自然元素

## 操作注意事项

### 性能平衡

1. **LOD使用**：远距离使用低细节模型
2. **实例化渲染**：植被使用GPU实例化
3. **遮挡剔除**：隐藏不可见物体

### 视觉一致性

1. **风格统一**：保持美术风格一致
2. **比例协调**：物体大小要合理
3. **色彩和谐**：使用统一的调色板

### 玩家体验

1. **导航清晰**：玩家能识别方向
2. **地标明显**：有明显的视觉地标
3. **引导自然**：通过环境引导玩家

## 常见错误

### 错误1：地形过于复杂

**问题**：地形网格过于密集，影响性能

**修正**：使用合理的网格密度，依赖纹理细节

### 错误2：植被分布不自然

**问题**：植被分布过于均匀，看起来不自然

**修正**：使用噪声算法实现自然分布

### 错误3：天气过渡生硬

**问题**：天气变化突然，没有过渡

**修正**：实现平滑的天气过渡效果

### 错误4：光照过曝或过暗

**问题**：场景光照不均匀，影响视觉效果

**修正**：调整光照强度和范围，使用环境光遮蔽

### 错误5：性能问题

**问题**：场景卡顿，帧率低

**修正**：使用LOD、遮挡剔除、实例化渲染优化性能
