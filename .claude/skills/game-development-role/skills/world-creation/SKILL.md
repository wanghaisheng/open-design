# World Creation

创建游戏世界，设计游戏世界的地理、历史、文化和规则，创造丰富完整的游戏世界。

## 适用场景

- 需要设计世界地理
- 需要设计世界历史
- 需要设计世界文化
- 需要设计世界规则
- 需要整合世界元素

## 操作步骤

### 步骤1：世界概念

**目标**：定义世界概念

**操作方法**：
1. 定义世界主题（世界观、核心主题、风格定位）
2. 确定世界风格（视觉风格、叙事风格、氛围风格）
3. 设计世界规模（世界大小、区域数量、复杂程度）

**检查清单**：
- [ ] 世界主题已定义
- [ ] 世界风格已确定
- [ ] 世界规模已设计

### 步骤2：地理设计

**目标**：设计世界地理

**操作方法**：
1. 设计世界地图（区域分布、边界设定）
2. 设计地形地貌（山脉、河流、平原、森林）
3. 设计气候环境（气候带、天气系统、季节变化）
4. 设计自然资源（资源分布、资源类型、资源获取）

**检查清单**：
- [ ] 世界地图已设计
- [ ] 地形地貌已设计
- [ ] 气候环境已设计
- [ ] 自然资源已设计

### 步骤3：历史设计

**目标**：设计世界历史

**操作方法**：
1. 设计世界历史（时间线、时代划分）
2. 设计重要事件（历史事件、转折点）
3. 设计历史人物（重要人物、角色设定）
4. 设计历史影响（历史对现在的影响、因果关系）

**检查清单**：
- [ ] 世界历史已设计
- [ ] 重要事件已设计
- [ ] 历史人物已设计
- [ ] 历史影响已设计

### 步骤4：文化设计

**目标**：设计世界文化

**操作方法**：
1. 设计社会结构（社会阶层、政治体系）
2. 设计文化传统（习俗、节日、礼仪）
3. 设计宗教信仰（宗教体系、信仰体系）
4. 设计价值观念（价值观、道德观）

**检查清单**：
- [ ] 社会结构已设计
- [ ] 文化传统已设计
- [ ] 宗教信仰已设计
- [ ] 价值观念已设计

### 步骤5：规则设计

**目标**：设计世界规则

**操作方法**：
1. 设计物理规则（物理法则、自然规律）
2. 设计魔法规则（魔法体系、魔法限制）
3. 设计社会规则（法律、习俗、禁忌）
4. 设计经济规则（货币、贸易、经济体系）

**检查清单**：
- [ ] 物理规则已设计
- [ ] 魔法规则已设计
- [ ] 社会规则已设计
- [ ] 经济规则已设计

### 步骤6：整合测试

**目标**：整合和测试世界

**操作方法**：
1. 整合世界元素（地理、历史、文化、规则）
2. 测试世界一致性（内部一致性、逻辑一致性）
3. 测试世界可玩性（玩法支持、探索支持、互动支持）

**检查清单**：
- [ ] 世界元素已整合
- [ ] 世界一致性已测试
- [ ] 世界可玩性已测试

## 常见错误

### 错误1：不定义世界概念

**问题**：不定义世界主题、不确定世界风格

**修正**：必须定义世界主题，确定世界风格，设计世界规模

### 错误2：不设计世界地理

**问题**：不设计世界地图、不设计地形地貌

**修正**：必须设计世界地图，设计地形地貌，设计气候环境

### 错误3：不设计世界历史

**问题**：不设计世界历史、不设计重要事件

**修正**：必须设计世界历史，设计重要事件，设计历史人物

### 错误4：不设计世界文化

**问题**：不设计社会结构、不设计文化传统

**修正**：必须设计社会结构，设计文化传统，设计宗教信仰

### 错误5：不测试世界一致性

**问题**：不测试世界一致性、不测试世界可玩性

**修正**：必须整合世界元素，测试世界一致性，测试世界可玩性
