# Phaser Development

Phaser 引擎开发技能，使用 Phaser 游戏框架进行 Web 游戏开发。

## 适用场景

- Phaser 项目设置
- TypeScript/JavaScript 脚本开发
- Phaser 场景管理
- Phaser 精灵和对象池
- Phaser 物理引擎集成

## 操作步骤

### 步骤1：创建 Phaser 项目

**目标**：创建和配置 Phaser 项目

**操作方法**：
1. 使用 Phaser CLI 创建项目
2. 配置项目结构
3. 设置 TypeScript 配置
4. 配置 Webpack 或 Vite
5. 运行开发服务器

**检查清单**：
- [ ] 项目创建成功
- [ ] TypeScript 配置正确
- [ ] 构建工具配置正确
- [ ] 开发服务器运行正常

### 步骤2：创建场景

**目标**：创建游戏场景

**操作方法**：
1. 创建场景类继承自 Phaser.Scene
2. 实现 preload 方法加载资源
3. 实现 create 方法创建游戏对象
4. 实现 update 方法处理游戏逻辑
5. 注册场景到游戏配置

**检查清单**：
- [ ] 场景类创建成功
- [ ] 资源加载正确
- [ ] 游戏对象创建正确
- [ ] 游戏逻辑实现正确
- [ ] 场景注册成功

### 步骤3：实现游戏对象

**目标**：创建和管理游戏对象

**操作方法**：
1. 创建精灵（Sprite）
2. 实现对象池
3. 配置物理属性
4. 实现动画
5. 添加交互功能

**检查清单**：
- [ ] 精灵创建成功
- [ ] 对象池实现正确
- [ ] 物理属性配置正确
- [ ] 动画实现正确
- [ ] 交互功能正常

### 步骤4：集成物理引擎

**目标**：使用 Phaser 物理引擎

**操作方法**：
1. 选择物理引擎（Arcade、Matter）
2. 配置物理世界
3. 创建物理体
4. 实现碰撞检测
5. 实现重力和其他物理效果

**检查清单**：
- [ ] 物理引擎选择正确
- [ ] 物理世界配置正确
- [ ] 物理体创建成功
- [ ] 碰撞检测正常
- [ ] 物理效果正常

### 步骤5：优化性能

**目标**：优化游戏性能

**操作方法**：
1. 使用对象池
2. 优化纹理压缩
3. 减少绘制调用
4. 优化粒子效果
5. 监控帧率

**检查清单**：
- [ ] 对象池实现正确

## 常见错误

### 错误1：资源加载失败
**问题**：资源路径错误或格式不支持
**修正**：检查资源路径，使用正确的资源格式

### 错误2：性能问题
**问题**：帧率下降，游戏卡顿
**修正**：使用对象池，优化渲染，减少物理计算

### 错误3：场景切换问题
**问题**：场景切换时资源未清理
**修正**：实现场景清理逻辑，正确管理资源生命周期
