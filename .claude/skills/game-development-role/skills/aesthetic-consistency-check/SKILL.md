# Aesthetic Consistency Check

美学一致性检查技能，检查美学决策的一致性。

## 适用场景

- 检查美学一致性
- 评估美学质量
- 识别美学问题
- 提供改进建议
- 维护美学标准

**目标**：建立美学一致性检查标准

**操作方法**：
1. 参考美学哲学和原则
2. 定义一致性维度
3. 设置检查阈值
4. 建立评分体系
5. 创建检查清单

**检查清单**：
- [ ] 哲学和原则已参考
- [ ] 一致性维度已定义
- [ ] 检查阈值已设置
- [ ] 评分体系已建立

## 操作注意事项

### 检查标准

1. **明确清晰**：检查标准要明确清晰
2. **可量化**：尽可能量化检查标准
3. **全面覆盖**：覆盖所有美学维度
4. **定期更新**：定期更新检查标准

### 检查方法

1. **系统化**：使用系统化的检查方法
2. **客观公正**：保持客观公正的态度
3. **细致全面**：细致全面地检查
4. **记录详细**：详细记录检查结果

### 问题处理

1. **优先级排序**：按严重性排序问题
2. **具体建议**：提供具体的改进建议
3. **可执行**：建议要可执行
4. **跟踪验证**：跟踪问题修复并验证

## 常见错误

### 错误1：检查标准不明确

**问题**：检查标准不明确，结果不一致

**修正**：建立明确清晰的检查标准

### 错误2：检查不全面

**问题**：检查不全面，遗漏问题

**修正**：使用检查清单确保全面检查

### 错误3：主观性强

**问题**：检查过于主观，结果不可靠

**修正**：使用量化标准，减少主观性

### 错误4：不跟踪修复

**问题**：发现问题后不跟踪修复

**修正**：建立问题跟踪机制

### 错误5：不提供具体建议

**问题**：只指出问题，不提供具体建议

**修正**：提供具体可执行的改进建议
