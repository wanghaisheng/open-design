# User Segmentation

用户分层技能，识别核心高价值用户并为他们设计专属内容和挑战。

## 适用场景

- 识别核心高价值用户
- 用户分层分析
- 精准优化设计
- 提升ARPU和LTV
- 避免普适性优化陷阱

## 操作步骤

### 步骤1：用户数据收集

**目标**：收集用户行为和付费数据

**操作方法**：
1. **行为数据收集**
   - 收集游戏时长数据
   - 收集关卡进度数据
   - 收集社交行为数据
   - 收集活动参与数据

2. **付费数据收集**
   - 收集付费金额数据
   - 收集付费频次数据
   - 收集付费内容数据
   - 收集付费时间数据

3. **留存数据收集**
   - 收集D1留存数据
   - 收集D7留存数据
   - 收集D30留存数据
   - 收集回归数据

**检查清单**：
- [ ] 行为数据已收集
- [ ] 付费数据已收集
- [ ] 留存数据已收集
- [ ] 数据已清洗

### 步骤2：用户分层分析

**目标**：将用户分层并识别核心高价值用户

**操作方法**：
1. **分层维度选择**
   - 按付费金额分层
   - 按游戏时长分层
   - 按付费频次分层
   - 按活跃度分层

2. **核心用户识别**
   - 识别前10%高价值用户
   - 分析高价值用户特征
   - 计算高价值用户贡献度
   - 识别高价值用户行为模式

3. **用户画像构建**
   - 构建核心用户画像
   - 分析核心用户偏好
   - 识别核心用户需求
   - 预测核心用户行为

**检查清单**：
- [ ] 分层维度已选择
- [ ] 核心用户已识别
- [ ] 用户画像已构建
- [ ] 特征已分析

### 步骤3：用户行为分析

**目标**：分析不同层级用户的行为差异

**操作方法**：
1. **行为对比分析**
   - 对比不同层级用户游戏时长
   - 对比不同层级用户关卡进度
   - 对比不同层级用户社交行为
   - 对比不同层级用户活动参与

2. **付费行为分析**
   - 分析不同层级用户付费偏好
   - 分析不同层级用户付费时机
   - 分析不同层级用户付费内容
   - 分析不同层级用户付费动机

3. **留存行为分析**
   - 分析不同层级用户留存差异
   - 分析流失用户特征
   - 分析留存用户特征
   - 识别流失风险点

**检查清单**：
- [ ] 行为对比已完成
- [ ] 付费行为已分析
- [ ] 留存行为已分析
- [ ] 差异已识别

### 步骤4：分层设计策略

**目标**：为不同层级用户设计差异化体验

**操作方法**：
1. **核心用户专属设计**
   - 设计专属游戏内容
   - 设计专属挑战
   - 设计专属奖励
   - 设计专属活动

2. **付费用户设计**
   - 设计付费专属内容
   - 设计付费专属优惠
   - 设计付费专属服务
   - 优化付费体验

3. **非付费用户设计**
   - 设计免费内容引导
   - 设计广告体验优化
   - 设计付费引导
   - 设计转化路径

**检查清单**：
- [ ] 核心用户设计已完成
- [ ] 付费用户设计已完成
- [ ] 非付费用户设计已完成
- [ ] 分层策略已制定

### 步骤5：效果评估与优化

**目标**：评估分层设计效果并持续优化

**操作方法**：
1. **效果评估**
   - 评估ARPU提升
   - 评估IAP收入提升
   - 评估留存变化
   - 评估用户满意度

2. **A/B测试**
   - 设计A/B测试方案
   - 执行A/B测试
   - 分析测试结果
   - 优化设计方案

3. **持续优化**
   - 持续监控数据
   - 持续优化设计
   - 持续调整策略
   - 持续提升效果

**检查清单**：
- [ ] 效果已评估
- [ ] A/B测试已完成
- [ ] 优化已实施
- [ ] 效果已验证

## 操作注意事项

### 分层原则

1. **数据驱动**：基于数据分层，避免主观判断
2. **动态调整**：定期调整分层标准
3. **目标明确**：分层要有明确目标
4. **可执行**：分层设计要可执行

### 设计原则

1. **精准优化**：避免普适性优化陷阱
2. **核心优先**：优先为核心用户设计
3. **体验平衡**：平衡不同层级用户体验
4. **转化引导**：设计非付费用户转化路径

### 优化原则

1. **持续监控**：持续监控关键数据
2. **A/B测试**：通过A/B测试优化
3. **快速迭代**：快速迭代优化
4. **效果验证**：验证优化效果

## 常见错误

### 错误1：普适性优化

**问题**：试图取悦所有用户，优化效果差

**修正**：精准优化，优先为核心用户设计

### 错误2：忽视非付费用户

**问题**：完全忽视非付费用户，失去转化机会

**修正**：为非付费用户设计转化路径

### 错误3：分层标准不合理

**问题**：分层标准不合理，分层结果不准确

**修正**：基于数据设计合理的分层标准

### 错误4：不持续优化

**问题**：分层设计后不持续优化

**修正**：持续监控数据，持续优化设计

### 错误5：体验失衡

**问题**：不同层级用户体验失衡

**修正**：平衡不同层级用户体验
