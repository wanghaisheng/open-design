# Environment Animation

环境动画技能，创建游戏环境的动态效果，包括植被动画、水体动画、天气效果和环境互动。

## 适用场景

- 创建植被风吹动画
- 实现水体流动和波浪效果
- 制作天气动画（雨、雪、雾）
- 实现环境互动动画（踩踏草地、水面涟漪）
- 优化环境动画性能

## 操作注意事项

### 视觉效果

1. **自然感**：动画要看起来自然，不夸张
2. **一致性**：所有环境动画风格要一致
3. **层次感**：不同距离的动画要有层次

### 性能考虑

1. **GPU加速**：尽可能使用GPU计算
2. **实例化**：相同动画使用实例化
3. **烘焙**：静态动画可以烘焙

### 玩家体验

1. **反馈及时**：互动动画要有即时反馈
2. **沉浸感强**：环境动画要增强沉浸感
3. **不干扰玩法**：动画不能干扰游戏玩法

## 常见错误

### 错误1：植被动画不自然

**问题**：植被摆动过于机械，不自然

**修正**：使用噪声函数，添加随机性

### 错误2：水体动画不真实

**问题**：水面效果不真实，缺乏细节

**修正**：使用法线贴图，添加反射折射

### 错误3：天气效果单调

**问题**：天气效果单一，缺乏变化

**修正**：添加多种天气类型，实现过渡

### 错误4：互动反馈不明显

**问题**：环境互动效果不明显

**修正**：增强视觉反馈，添加音效

### 错误5：性能问题

**问题**：环境动画占用过多资源

**修正**：使用LOD、GPU计算、实例化优化性能
