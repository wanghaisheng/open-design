# Market Insight

市场洞察技能，超越榜单分析市场趋势，通过广告素材和自媒体捕捉早期趋势。

## 适用场景

- 捕捉市场早期趋势
- 分析广告投放素材
- 监控自媒体生态
- 识别融合机会
- 指导立项决策

## 操作步骤

### 步骤1：超越榜单分析

**目标**：分析榜单之外的市场趋势

**操作方法**：
1. **榜单局限性认知**
   - 理解榜单反映过去结果
   - 认识榜单滞后性
   - 寻找领先指标
   - 关注新兴品类

2. **广告素材分析**
   - 分析近30天广告投放素材
   - 拆解玩法和创意
   - 分析变现模式
   - 识别跑量素材

3. **数据平台辅助**
   - 使用Data.ai分析数据
   - 使用Sensor Tower追踪趋势
   - 分析新兴品类
   - 识别机会点

**检查清单**：
- [ ] 榜单局限性已认知
- [ ] 广告素材已分析
- [ ] 数据平台已使用
- [ ] 趋势已识别

### 步骤2：监控自媒体生态

**目标**：通过自媒体平台捕捉早期趋势

**操作方法**：
1. **平台监控**
   - 微信公众号：国内图文内容
   - B站：国内视频内容
   - YouTube：国际视频内容
   - Instagram：图片和短视频
   - TikTok：短视频内容

2. **特定账号关注**
   - 关注@storeglide
   - 关注@creativespyofc
   - 关注行业KOL
   - 关注发行商账号

3. **内容分析**
   - 分析内容更新频率
   - 识别新兴话题
   - 分析互动数据
   - 提取趋势信号

**检查清单**：
- [ ] 平台已监控
- [ ] 特定账号已关注
- [ ] 内容已分析
- [ ] 趋势信号已提取

### 步骤3：寻找融合机会

**目标**：识别新颖视觉与成熟玩法的融合机会

**操作方法**：
1. **融合公式应用**
   - 新颖视觉/主题 + 成熟玩法 = 新产品
   - 识别新颖视觉元素
   - 识别成熟玩法
   - 设计融合方案

2. **案例分析**
   - 《Unravel Master》：毛线元素 + Screw Master
   - 分析融合逻辑
   - 提取融合模式
   - 应用到新创意

3. **创意生成**
   - 基于融合公式生成创意
   - 评估创意可行性
   - 优先排序创意
   - 选择最佳方案

**检查清单**：
- [ ] 融合公式已应用
- [ ] 案例已分析
- [ ] 创意已生成
- [ ] 方案已选择

### 步骤4：趋势验证

**目标**：验证识别的趋势是否真实可靠

**操作方法**：
1. **多源验证**
   - 从多个来源验证趋势
   - 对比不同平台数据
   - 分析趋势一致性
   - 评估趋势可靠性

2. **数据验证**
   - 分析搜索趋势
   - 分析下载趋势
   - 分析收入趋势
   - 分析用户讨论

3. **专家验证**
   - 咨询行业专家
   - 参考行业报告
   - 参加行业会议
   - 获取内部信息

**检查清单**：
- [ ] 多源验证已完成
- [ ] 数据验证已完成
- [ ] 专家验证已完成
- [ ] 趋势已确认

### 步骤5：趋势应用

**目标**：将市场洞察应用到立项决策

**操作方法**：
1. **立项建议**
   - 基于趋势提出立项建议
   - 评估项目可行性
   - 分析市场机会
   - 评估竞争情况

2. **设计指导**
   - 基于趋势指导设计
   - 确定美术风格
   - 确定玩法方向
   - 确定变现模式

3. **风险评估**
   - 评估趋势风险
   - 评估竞争风险
   - 评估技术风险
   - 制定风险应对

**检查清单**：
- [ ] 立项建议已提出
- [ ] 设计指导已完成
- [ ] 风险评估已完成
- [ ] 决策支持已提供

## 操作注意事项

### 趋势识别

1. **早期识别**：关注早期趋势信号
2. **多源验证**：从多个来源验证趋势
3. **区分噪音**：区分真实趋势和噪音
4. **持续监控**：持续监控趋势变化

### 数据分析

1. **数据质量**：确保数据质量可靠
2. **数据对比**：对比不同来源数据
3. **趋势分析**：分析趋势变化
4. **因果分析**：分析因果关系

### 应用决策

1. **谨慎决策**：基于充分信息决策
2. **风险评估**：全面评估风险
3. **快速响应**：快速响应市场变化
4. **持续跟踪**：持续跟踪趋势发展

## 常见错误

### 错误1：只看榜单

**问题**：只关注榜单，忽视早期趋势

**修正**：超越榜单，关注广告素材和自媒体

### 错误2：数据误读

**问题**：误读数据，得出错误结论

**修正**：确保数据理解正确，多源验证

### 错误3：趋势误判

**问题**：将噪音误认为趋势

**修正**：多源验证，区分真实趋势和噪音

### 错误4：应用不当

**问题**：趋势应用不当，与项目不匹配

**修正**：确保趋势与项目匹配，谨慎应用

### 错误5：不持续监控

**问题**：不持续监控，错过趋势变化

**修正**：建立持续监控机制，及时响应变化
