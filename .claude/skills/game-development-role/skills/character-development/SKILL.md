# Character Development

角色发展，设计游戏角色的成长轨迹、动机和关系网络。

## 适用场景

- 角色背景故事设计
- 角色动机和目标定义
- 角色性格特征创建
- 角色成长轨迹规划
- 角色关系网络设计

## 操作步骤

### 步骤1：设计角色背景

**目标**：创建角色的背景故事

**操作方法**：
1. 定义角色的出身和成长经历
2. 识别形成角色的关键事件
3. 确定影响角色的重要人物
4. 建立背景与当前状态的关联

**检查清单**：
- [ ] 背景故事完整
- [ ] 关键事件有意义
- [ ] 重要人物影响合理
- [ ] 与当前状态相关

### 步骤2：定义角色动机

**目标**：明确角色的驱动力

**操作方法**：
1. 确定内在动机（内心驱动力）
2. 确定外在动机（外部影响因素）
3. 识别冲突动机（相互冲突的驱动力）
4. 确保动机与行为一致

**检查清单**：
- [ ] 内在动机清晰
- [ ] 外在动机明确
- [ ] 冲突动机合理
- [ ] 动机与行为一致

### 步骤3：设计角色成长

**目标**：规划角色的成长轨迹

**操作方法**：
1. 定义角色的初始状态
2. 设计成长挑战
3. 确定转变点
4. 定义角色的最终状态

**检查清单**：
- [ ] 初始状态明确
- [ ] 成长挑战有意义
- [ ] 转变点关键
- [ ] 最终状态合理

### 步骤4：设计角色关系

**目标**：创建角色关系网络

**操作方法**：
1. 识别角色之间的关系
2. 定义关系类型和强度
3. 设计关系发展轨迹
4. 确保关系发展的连贯性

**检查清单**：
- [ ] 关系网络完整
- [ ] 关系类型清晰
- [ ] 关系强度合理
- [ ] 发展轨迹连贯

## 操作注意事项

### 一致性
- 保持角色行为一致
- 确保角色动机合理
- 维持角色关系连贯

### 深度
- 创建多层次的角色
- 设计复杂的动机
- 赋予角色内在冲突

### 相关性
- 角色与故事相关
- 角色与玩家相关
- 角色与游戏玩法相关

## 输出格式

### 角色档案
- 基本信息：姓名、年龄、背景
- 性格特征：主要性格特点
- 动机目标：内在和外在动机
- 成长弧线：角色发展轨迹

### 关系图
- 角色关系网络
- 关系类型和强度
- 关系发展轨迹

### 对话风格
- 说话方式
- 语言特点
- 对话示例

## 协作协议

### 向谁汇报
- narrative-director：叙事方向
- creative-director：创意方向

### 协调对象
- writer：角色对话和描述
- character-artist：角色视觉设计
- level-designer：角色在关卡中的实现

## 常见错误

### 错误1：角色动机不清晰
**问题**：角色的行为缺乏合理的动机
**修正**：明确内在和外在动机

### 错误2：角色发展不连贯
**问题**：角色成长轨迹缺乏逻辑
**修正**：确保转变点有意义

### 错误3：角色关系过于复杂
**问题**：关系网络过于复杂，难以理解
**修正**：简化关系网络，聚焦关键关系
