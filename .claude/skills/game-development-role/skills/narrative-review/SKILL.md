# Narrative Review

叙事评审，评审叙事内容的质量、一致性和影响力。

## 适用场景

- 叙事内容质量评估
- 故事结构和节奏检查
- 角色塑造和对话评估
- 叙事一致性验证
- 叙事影响力评估

## 操作步骤

### 步骤1：准备评审

**目标**：收集和理解叙事材料

**操作方法**：
1. 收集所有叙事材料
2. 了解故事背景和目标
3. 确定评审标准
4. 准备评审检查清单

**检查清单**：
- [ ] 叙事材料完整
- [ ] 背景和目标明确
- [ ] 评审标准确定
- [ ] 检查清单准备

### 步骤2：评估故事结构

**目标**：检查故事结构的完整性

**操作方法**：
1. 检查结构完整性
2. 评估节奏控制
3. 验证情节连贯性
4. 评估高潮和结局效果

**检查清单**：
- [ ] 结构完整
- [ ] 节奏合理
- [ ] 情节连贯
- [ ] 高潮有力
- [ ] 结局满意

### 步骤3：检查角色塑造

**目标**：评估角色质量和深度

**操作方法**：
1. 评估角色深度
2. 检查角色动机合理性
3. 验证角色发展轨迹
4. 评估对话质量

**检查清单**：
- [ ] 角色有深度
- [ ] 动机合理
- [ ] 发展轨迹清晰
- [ ] 对话自然

### 步骤4：验证一致性

**目标**：确保叙事内部一致

**操作方法**：
1. 检查内部逻辑一致性
2. 验证角色行为一致性
3. 检查世界观一致性
4. 验证时间线连贯性

**检查清单**：
- [ ] 内部逻辑一致
- [ ] 角色行为一致
- [ ] 世界观一致
- [ ] 时间线连贯

### 步骤5：评估影响力

**目标**：评估叙事的情感冲击和参与度

**操作方法**：
1. 评估情感冲击力
2. 检查玩家参与度
3. 验证主题共鸣
4. 评估长期记忆价值

**检查清单**：
- [ ] 情感冲击强
- [ ] 参与度高
- [ ] 主题共鸣
- [ ] 长期记忆价值

### 步骤6：提供反馈

**目标**：提供具体可行的改进建议

**操作方法**：
1. 整理评审发现
2. 提供具体建议
3. 按优先级排序
4. 跟踪改进进展

**检查清单**：
- [ ] 发现具体
- [ ] 建议可行
- [ ] 优先级明确
- [ ] 跟踪机制建立

## 操作注意事项

### 客观评审
- 基于标准而非个人偏好
- 提供具体证据
- 区分问题和偏好

### 建设性反馈
- 专注于可改进的方面
- 提供可行的建议
- 平衡批评和表扬

### 持续改进
- 定期评审叙事内容
- 跟踪改进效果
- 学习优秀案例

## 输出格式

### 评审报告
- 整体评估
- 具体发现
- 改进建议
- 优先级排序

### 问题清单
- 一致性问题
- 质量问题
- 改进机会

### 反馈摘要
- 主要优点
- 主要问题
- 行动建议

## 评审标准

### 质量标准
- 故事结构完整
- 角色塑造深入
- 对话自然流畅
- 主题表达清晰

### 一致性标准
- 内部逻辑一致
- 角色行为一致
- 世界观一致
- 时间线连贯

### 影响力标准
- 情感冲击力
- 玩家参与度
- 主题共鸣
- 长期记忆

## 协作协议

### 向谁汇报
- narrative-director：叙事方向
- creative-director：创意方向

### 协调对象
- writer：叙事内容修改
- level-designer：关卡中的叙事实现
- world-builder：世界中的叙事元素

## 常见错误

### 错误1：评审过于主观
**问题**：基于个人偏好而非标准进行评审
**修正**：使用明确的评审标准，提供具体证据

### 错误2：反馈不够具体
**问题**：反馈过于笼统，难以执行
**修正**：提供具体可行的改进建议

### 错误3：忽视优点
**问题**：只关注问题，忽视优点
**修正**：平衡批评和表扬，保持建设性
