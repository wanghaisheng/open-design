# Character Art

创建游戏角色美术，包括角色模型、纹理、绑定和动画，创造生动的游戏角色。

## 角色要素

### 1. 角色模型 (Character Models)
- 高模：高精度模型
- 低模：游戏用模型
- 拓扑：模型拓扑
- UV映射：UV展开

### 2. 纹理材质 (Textures & Materials)
- 漫反射：基础颜色
- 法线：法线贴图
- 粗糙度：粗糙度贴图
- 金属度：金属度贴图

### 3. 角色绑定 (Character Rigging)
- 骨骼系统：骨骼结构
- 皮肤绑定：权重绘制
- 控制器：控制器设置
- IK/FK：IK/FK切换

### 4. 角色动画 (Character Animation)
- 动画剪辑：动画片段
- 动画状态机：状态机
- 动画混合：动画混合
- 动画压缩：动画压缩

## 创作工具

### 3D建模
- ZBrush：高模雕刻
- Blender：建模和绑定
- Maya：专业绑定
- 3ds Max：专业建模

### 纹理制作
- Substance Painter：角色纹理
- Substance Designer：程序化纹理
- Mari：高精度纹理
- Photoshop：后期处理

### 动画工具
- Maya：专业动画
- Blender：动画制作
- MotionBuilder：动作捕捉
- 3ds Max：动画制作

## 创作原则

### 1. 角色识别
- 独特外观
- 清晰轮廓
- 角色个性

### 2. 性能优化
- 模型优化
- 纹理优化
- 动画优化

### 3. 可动画性
- 良好的拓扑
- 合理的绑定
- 流畅的动画

### 4. 风格一致
- 风格统一
- 比例一致
- 细节一致
