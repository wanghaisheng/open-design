# Character Art

创建游戏角色美术，包括角色模型、纹理、绑定和动画，创造生动的游戏角色。

## 适用场景

- 需要创建角色模型
- 需要制作角色纹理
- 需要进行角色绑定
- 需要制作角色动画
- 需要优化角色性能

## 操作步骤

### 步骤1：概念设计

**目标**：理解角色需求和风格方向

**操作方法**：
1. 参考概念图（concept-art技能输出）
2. 理解角色需求（功能、玩法需求）
3. 确定风格方向（写实、卡通、像素等）
4. 识别技术限制（多边形数量、纹理分辨率）

**检查清单**：
- [ ] 概念图已参考
- [ ] 角色需求已理解
- [ ] 风格方向已确定
- [ ] 技术限制已识别

### 步骤2：高模雕刻

**目标**：创建高精度模型

**操作方法**：
1. 使用 ZBrush 进行高模雕刻
2. 雕刻细节（肌肉、衣物、装备）
3. 输出高模

**检查清单**：
- [ ] 高模已雕刻
- [ ] 细节已添加
- [ ] 高模已输出

### 步骤3：低模制作

**目标**：创建游戏用模型

**操作方法**：
1. 制作拓扑（使用 Blender/Maya/3ds Max）
2. UV展开
3. 输出低模

**检查清单**：
- [ ] 拓扑已制作
- [ ] UV已展开
- [ ] 低模已输出

### 步骤4：纹理制作

**目标**：创建角色纹理

**操作方法**：
1. 烘焙贴图（从高模到低模）
2. 使用 Substance Painter 绘制纹理
3. 制作材质（漫反射、法线、粗糙度、金属度）
4. 使用 Photoshop 后期处理

**检查清单**：
- [ ] 贴图已烘焙
- [ ] 纹理已绘制
- [ ] 材质已制作
- [ ] 后期处理已完成

### 步骤5：角色绑定

**目标**：创建角色骨骼和控制器

**操作方法**：
1. 创建骨骼系统（使用 Blender/Maya）
2. 皮肤绑定（权重绘制）
3. 设置控制器
4. 配置 IK/FK 切换

**检查清单**：
- [ ] 骨骼系统已创建
- [ ] 皮肤绑定已完成
- [ ] 控制器已设置
- [ ] IK/FK已配置

### 步骤6：动画制作

**目标**：创建角色动画

**操作方法**：
1. 制作动画剪辑（使用 Maya/Blender/MotionBuilder）
2. 设置动画状态机
3. 配置动画混合
4. 动画压缩优化

**检查清单**：
- [ ] 动画剪辑已制作
- [ ] 动画状态机已设置
- [ ] 动画混合已配置
- [ ] 动画已压缩

### 步骤7：性能优化

**目标**：优化角色性能

**操作方法**：
1. 优化模型（减少多边形）
2. 优化纹理（压缩纹理）
3. 优化动画（减少关键帧）

**检查清单**：
- [ ] 模型已优化
- [ ] 纹理已优化
- [ ] 动画已优化

### 步骤8：输出角色资产

**目标**：输出最终角色资产

**操作方法**：
1. 输出3D模型文件（FBX/GLTF）
2. 输出纹理文件（PNG/TGA）
3. 输出绑定文件
4. 输出动画文件

**检查清单**：
- [ ] 模型文件已输出
- [ ] 纹理文件已输出
- [ ] 绑定文件已输出
- [ ] 动画文件已输出

## 操作注意事项

### 角色识别

1. **独特外观**：确保角色有独特的外观
2. **清晰轮廓**：确保角色有清晰的轮廓
3. **角色个性**：确保角色体现个性

### 性能优化

1. **模型优化**：减少多边形数量
2. **纹理优化**：压缩纹理，减少内存占用
3. **动画优化**：减少关键帧，优化动画数据

### 可动画性

1. **良好的拓扑**：确保拓扑适合动画
2. **合理的绑定**：确保绑定合理，易于动画
3. **流畅的动画**：确保动画流畅自然

### 风格一致

1. **风格统一**：确保与游戏整体风格统一
2. **比例一致**：确保角色比例一致
3. **细节一致**：确保细节风格一致

- **character-designer**：角色设计和背景
- **technical-artist**：技术实现和优化

## 常见错误

### 错误1：忽略技术限制

**问题**：模型多边形或纹理分辨率超出技术限制

**修正**：在设计前明确技术限制，确保在限制内

### 错误2：拓扑不适合动画

**问题**：拓扑结构不适合动画，导致变形问题

**修正**：确保拓扑适合动画，使用合理的布线

### 错误3：纹理不优化

**问题**：纹理未优化，占用过多内存

**修正**：必须优化纹理，压缩纹理格式

### 错误4：绑定不合理

**问题**：绑定不合理，难以动画

**修正**：确保绑定合理，设置适当的控制器

### 错误5：风格不一致

**问题**：角色风格与游戏整体风格不一致

**修正**：确保角色风格与游戏整体风格一致
