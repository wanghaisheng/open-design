# Environment Art

创建游戏环境美术，包括场景模型、纹理、光照和环境特效，营造沉浸式游戏世界。

## 环境要素

### 1. 场景模型 (Scene Models)
- 建筑模型：建筑物、结构
- 自然模型：树木、岩石、水体
- 道具模型：场景道具
- 地形模型：地形地貌

### 2. 纹理材质 (Textures & Materials)
- 漫反射纹理：基础颜色纹理
- 法线纹理：法线贴图
- 粗糙度纹理：粗糙度贴图
- 金属度纹理：金属度贴图

### 3. 光照系统 (Lighting System)
- 直接光照：太阳光、点光源
- 间接光照：环境光、全局光照
- 阴影：阴影映射、软阴影
- 光照贴图：光照烘焙

### 4. 环境特效 (Environmental Effects)
- 天空盒：天空、云层
- 天气系统：雨、雪、雾
- 粒子效果：灰尘、烟雾
- 后处理：色调映射、泛光

### 1. 视觉一致性
- 风格一致
- 比例一致
- 色彩一致

### 2. 性能优化
- 模型优化
- 纹理优化
- LOD设计

### 3. 沉浸感
- 细节丰富
- 氛围营造

### 4. 可重用性
