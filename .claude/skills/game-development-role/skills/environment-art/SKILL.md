# Environment Art

创建游戏环境美术，包括场景模型、纹理、光照和环境特效，营造沉浸式游戏世界。

## 适用场景

- 需要创建场景模型
- 需要制作纹理材质
- 需要设置光照系统
- 需要添加环境特效
- 需要优化环境性能

## 操作步骤

### 步骤1：概念设计

**目标**：理解环境需求和风格方向

**操作方法**：
1. 参考概念图（concept-art技能输出）
2. 理解环境需求（功能、玩法需求）
3. 确定风格方向（写实、卡通、像素等）

**检查清单**：
- [ ] 概念图已参考
- [ ] 环境需求已理解
- [ ] 风格方向已确定

### 步骤2：白盒搭建

**目标**：搭建白盒测试空间

**操作方法**：
1. 搭建白盒（使用 Blender/Maya/3ds Max）
2. 测试空间（测试空间布局）
3. 调整比例（调整比例关系）

**检查清单**：
- [ ] 白盒已搭建
- [ ] 空间已测试
- [ ] 比例已调整

### 步骤3：资产生产

**目标**：生产环境资产

**操作方法**：
1. 建模（建筑模型、自然模型、道具模型、地形模型）
2. 纹理（使用 Substance Painter/Substance Designer/Photoshop/Quixel）
3. 材质（漫反射纹理、法线纹理、粗糙度纹理、金属度纹理）

**检查清单**：
- [ ] 模型已建模
- [ ] 纹理已制作
- [ ] 材质已创建

### 步骤4：场景组装

**目标**：组装场景

**操作方法**：
1. 组装场景（组装所有资产）
2. 添加细节（添加细节元素）
3. 调整布局（调整场景布局）

**检查清单**：
- [ ] 场景已组装
- [ ] 细节已添加
- [ ] 布局已调整

### 步骤5：光照设置

**目标**：设置光照系统

**操作方法**：
1. 设置光照（直接光照、间接光照）
2. 调整阴影（阴影映射、软阴影）
3. 烘焙光照（光照贴图）

**检查清单**：
- [ ] 光照已设置
- [ ] 阴影已调整
- [ ] 光照已烘焙

### 步骤6：环境特效

**目标**：添加环境特效

**操作方法**：
1. 天空盒（天空、云层）
2. 天气系统（雨、雪、雾）
3. 粒子效果（灰尘、烟雾）
4. 后处理（色调映射、泛光）

**检查清单**：
- [ ] 天空盒已设置
- [ ] 天气系统已设置
- [ ] 粒子效果已添加
- [ ] 后处理已设置

### 步骤7：性能优化

**目标**：优化环境性能

**操作方法**：
1. 优化模型（减少多边形）
2. 优化纹理（压缩纹理）
3. 优化光照（优化光照贴图）

**检查清单**：
- [ ] 模型已优化
- [ ] 纹理已优化
- [ ] 光照已优化

## 常见错误

### 错误1：不搭建白盒

**问题**：不搭建白盒，直接建模

**修正**：必须先搭建白盒，测试空间和比例

### 错误2：不进行性能优化

**问题**：不优化模型、纹理、光照

**修正**：必须优化模型、纹理、光照

### 错误3：风格不一致

**问题**：环境风格与游戏整体风格不一致

**修正**：确保风格一致，符合游戏整体风格

### 错误4：不设置光照

**问题**：不设置光照系统，环境缺乏氛围

**修正**：必须设置光照系统，调整阴影，烘焙光照

### 错误5：不添加环境特效

**问题**：不添加环境特效，环境缺乏细节

**修正**：必须添加环境特效（天空盒、天气系统、粒子效果、后处理）
