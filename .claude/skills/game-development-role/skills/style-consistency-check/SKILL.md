# Style Consistency Check

检查视觉风格一致性，确保所有视觉资产符合统一的视觉风格和艺术方向。

## 检查要素

### 1. 色彩一致性 (Color Consistency)
- 色彩方案：检查色彩方案一致性
- 色彩使用：检查色彩使用一致性
- 色彩平衡：检查色彩平衡一致性
- 色彩情感：检查色彩情感一致性

### 2. 造型一致性 (Style Consistency)
- 造型风格：检查造型风格一致性
- 线条风格：检查线条风格一致性
- 比例关系：检查比例关系一致性
- 细节处理：检查细节处理一致性

### 3. 氛围一致性 (Atmosphere Consistency)
- 氛围营造：检查氛围营造一致性
- 光照风格：检查光照风格一致性
- 材质表现：检查材质表现一致性
- 整体感觉：检查整体感觉一致性

### 4. 质量一致性 (Quality Consistency)
- 细节质量：检查细节质量一致性
- 完成度：检查完成度一致性
- 技术质量：检查技术质量一致性
- 艺术质量：检查艺术质量一致性

## 检查原则

### 1. 严格性
- 严格检查标准
- 严格执行规则
- 严格质量要求

### 2. 一致性
- 检查标准一致
- 检查方法一致
- 检查结果一致

### 3. 及时性
- 及时检查
- 及时反馈
- 及时修正

### 4. 可追溯性
- 记录检查结果
- 记录检查过程
- 记录修正历史

## 检查流程

### 1. 建立标准
- 建立风格标准
- 建立检查规则
- 建立质量基准

### 2. 执行检查
- 检查色彩一致性
- 检查造型一致性
- 检查氛围一致性
- 检查质量一致性

### 3. 问题记录
- 记录不一致问题
- 记录质量问题
- 记录修正建议

### 4. 反馈修正
- 提供检查报告
- 提供修正建议
- 跟踪修正进度

## 输出格式

### 检查报告
- 一致性检查结果
- 问题列表
- 修正建议
- 质量评估
