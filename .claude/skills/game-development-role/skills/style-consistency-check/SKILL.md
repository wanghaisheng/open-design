# Style Consistency Check

检查视觉风格一致性，确保所有视觉资产符合统一的视觉风格和艺术方向。

## 适用场景

- 需要检查色彩一致性
- 需要检查造型一致性
- 需要检查氛围一致性
- 需要检查质量一致性
- 需要跟踪修正进度

## 操作步骤

### 步骤1：建立标准

**目标**：建立风格标准

**操作方法**：
1. 建立风格标准（色彩方案、造型风格、氛围风格）
2. 建立检查规则（检查标准、检查方法）
3. 建立质量基准（质量标准、质量要求）

**检查清单**：
- [ ] 风格标准已建立
- [ ] 检查规则已建立
- [ ] 质量基准已建立

### 步骤2：执行检查

**目标**：执行一致性检查

**操作方法**：
1. 检查色彩一致性（色彩方案、色彩使用、色彩平衡、色彩情感）
2. 检查造型一致性（造型风格、线条风格、比例关系、细节处理）
3. 检查氛围一致性（氛围营造、光照风格、材质表现、整体感觉）
4. 检查质量一致性（细节质量、完成度、技术质量、艺术质量）

**检查清单**：
- [ ] 色彩一致性已检查
- [ ] 造型一致性已检查
- [ ] 氛围一致性已检查
- [ ] 质量一致性已检查

### 步骤3：问题记录

**目标**：记录不一致问题

**操作方法**：
1. 记录不一致问题（问题描述、问题位置）
2. 记录质量问题（质量缺陷、质量偏差）
3. 记录修正建议（修正方法、修正优先级）

**检查清单**：
- [ ] 不一致问题已记录
- [ ] 质量问题已记录
- [ ] 修正建议已记录

### 步骤4：反馈修正

**目标**：提供反馈并跟踪修正

**操作方法**：
1. 提供检查报告（检查结果、问题列表）
2. 提供修正建议（修正方案、修正时间）
3. 跟踪修正进度（修正状态、修正验证）

**检查清单**：
- [ ] 检查报告已提供
- [ ] 修正建议已提供
- [ ] 修正进度已跟踪

## 输出格式

### 检查报告
- 一致性检查结果
- 问题列表
- 修正建议
- 质量评估

## 常见错误

### 错误1：不建立风格标准

**问题**：不建立风格标准、不建立检查规则

**修正**：必须建立风格标准，建立检查规则，建立质量基准

### 错误2：不执行一致性检查

**问题**：不检查色彩一致性、不检查造型一致性

**修正**：必须检查色彩一致性，造型一致性，氛围一致性，质量一致性

### 错误3：不记录不一致问题

**问题**：不记录不一致问题、不记录质量问题

**修正**：必须记录不一致问题，记录质量问题，记录修正建议

### 错误4：不跟踪修正进度

**问题**：不提供检查报告、不跟踪修正进度

**修正**：必须提供检查报告，提供修正建议，跟踪修正进度

### 错误5：检查不全面

**问题**：不全面检查风格、不全面检查质量

**修正**：必须全面检查风格，全面检查质量，全面检查一致性
