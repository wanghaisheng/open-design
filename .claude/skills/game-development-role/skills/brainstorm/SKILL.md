# Brainstorm

引导游戏概念创意生成 - 从零想法到结构化游戏概念文档。使用专业工作室创意技巧、玩家心理框架和结构化创意探索。

## 创意要素

### 1. 创意发现 (Creative Discovery)
- 情感锚点：游戏中打动玩家的时刻
- 品味档案：玩家游戏偏好
- 实际约束：团队规模、时间、平台
- 创意简报：情感目标、品味、约束总结

### 2. 概念生成 (Concept Generation)
- 动词优先设计：从核心动词开始
- 混搭方法：结合两个意外元素
- 体验优先设计：从情感目标倒推
- 概念呈现：标题、电梯演讲、核心动词

### 3. 核心循环设计 (Core Loop Design)
- 30秒循环：时刻到时刻的玩法
- 5分钟循环：短期目标结构
- 会话循环：完整游戏会话
- 进度循环：长期成长目标

### 4. 支柱和边界 (Pillars and Boundaries)
- 游戏支柱：3-5个设计支柱
- 反支柱：游戏不做什么
- 设计测试：支柱决策测试
- 张力平衡：支柱间的张力

## 创意原则

### 1. 无判断
- 创意探索期不判断
- 鼓励非常规想法
- 建立而非否定

### 2. 约束作为燃料
- 约束激发创意
- 限制产生最佳想法
- 时间框定阶段

### 3. 协作探索
- 用户驱动协作
- AI作为创意促进者
- 逐步推进决策

### 4. 结构化方法
- 使用专业框架
- MDA框架
- 自我决定理论

## 创意流程

### 阶段1：创意发现
- 理解人而非游戏
- 情感锚点问题
- 品味档案分析
- 实际约束界定
- 生成创意简报

### 阶段2：概念生成
- 动词优先设计
- 混搭方法
- 体验优先设计
- 生成3个不同概念
- 用户选择概念

### 阶段3：核心循环设计
- 30秒循环分析
- 5分钟循环结构
- 会话循环定义
- 进度循环设计
- 玩家动机分析

### 阶段4：支柱和边界
- 定义3-5个支柱
- 定义反支柱
- 设计测试
- 平衡张力

### 阶段5：玩家类型验证
- Bartle分类
- Quantic Foundry动机模型
- 目标玩家定义
- 市场验证

### 阶段6：范围和可行性
- 引擎推荐
- 美术管线
- 内容范围
- MVP定义
- 最大风险
