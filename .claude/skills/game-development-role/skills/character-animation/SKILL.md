# Character Animation

角色动画技能，创建游戏角色的动画，包括运动、战斗、表情和交互动画。

## 适用场景

- 创建角色运动动画（行走、跑步、跳跃）
- 设计战斗动画（攻击、受击、死亡）
- 实现表情动画（面部表情、口型）
- 制作交互动画（拾取、使用、攀爬）
- 优化动画性能和过渡

## 操作步骤

### 步骤1：运动动画

**目标**：创建角色基础运动动画

**操作方法**：
1. 设计运动周期（行走、跑步、蹲伏）
2. 制作关键帧动画
3. 实现动画混合（走路到跑步过渡）
4. 添加身体细节动画（手臂摆动、头部转动）
5. 优化动画循环点

## 操作注意事项

### 动画质量

1. **遵循物理规律**：动画要符合物理规律
2. **重量感**：体现角色的重量和惯性
3. **流畅性**：动画过渡要流畅自然

### 性能考虑

1. **骨骼数量**：控制骨骼数量在合理范围
2. **关键帧密度**：合理分布关键帧
3. **动画压缩**：使用压缩减少内存占用

### 玩家体验

1. **响应速度**：攻击动画要有快速响应
2. **反馈清晰**：受击动画要有清晰反馈
3. **表现力**：表情要能传达情感

## 常见错误

### 错误1：动画不流畅

**问题**：动画过渡生硬，有卡顿

**修正**：优化过渡曲线，增加过渡时间

### 错误2：缺乏重量感

**问题**：角色动作轻飘飘，没有重量感

**修正**：调整动画曲线，添加预备和缓冲

### 错误3：表情僵硬

**问题**：面部表情不自然，僵硬

**修正**：增加面部细节，优化混合权重

### 错误4：IK不准确

**问题**：IK计算不准确，脚部穿模

**修正**：调整IK参数，添加约束

### 错误5：性能问题

**问题**：动画占用过多资源

**修正**：优化骨骼和关键帧，使用动画压缩
