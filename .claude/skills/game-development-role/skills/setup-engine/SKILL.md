# Setup Engine

配置项目游戏引擎和版本。在 CLAUDE.md 中固定引擎，检测知识差距，当版本超出 LLM 训练数据时通过 WebSearch 填充引擎参考文档。

## 适用场景

- 需要配置游戏引擎
- 需要查找引擎版本
- 需要更新技术栈
- 需要填充参考文档
- 需要检测知识差距

## 常见错误

### 错误1：不解析参数

**问题**：不检查参数格式、不确定配置模式

**修正**：必须检查参数格式，确定配置模式，设置配置目标

### 错误2：不查找引擎版本

**问题**：不使用 WebSearch 查找、不确认最新版本

**修正**：必须使用 WebSearch 查找，确认最新稳定版本，用户确认版本

### 错误3：不更新技术栈

**问题**：不读取 CLAUDE.md、不更新技术栈部分

**修正**：必须读取 CLAUDE.md，更新技术栈部分，填充实际值

### 错误4：不检测知识差距

**问题**：不检查训练数据截止、不评估风险级别

**修正**：必须检查训练数据截止，比较版本覆盖，评估风险级别

### 错误5：不填充参考文档

**问题**：不创建参考文档、不搜索迁移指南

**修正**：必须创建参考文档，搜索迁移指南，搜索破坏性更改
